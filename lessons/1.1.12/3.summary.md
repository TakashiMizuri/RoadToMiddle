# Methods — краткая выжимка

> `1.1.12` · ~8 мин на повторение

## Главная идея

**Methods** группируют behavior за именем с **parameters** и **return type**. **Overloading** — одно имя, разные signatures; compiler выбирает match at compile time. Extract из `Main` — первый шаг к maintainable console и service code.

## Ключевые пункты

- Signature = **name + parameter types** (не return type alone).
- **`void`** — нет value; **`return;`** для early exit.
- **Expression-bodied**: `Type Name() => expr;`
- **Overloading**: разное число/типы parameters; **same concept**.
- **Optional parameters** с `= default` — только trailing.
- Illegal: два overload **только** по return type.
- PascalCase для method names.

## Самые важные примеры

```csharp
static int Add(int a, int b) => a + b;

static int Area(int side) => side * side;
static int Area(int w, int h) => w * h;
```

```csharp
static int ReadIntPrompt(string message)
{
    while (true)
        if (int.TryParse(Console.ReadLine(), out int n)) return n;
}
```

## Частые ошибки

- God **Main** без extraction.
- Overload для **разных** операций под одним name.
- Missing **return** на some path (CS0161).

## Мнемоника

> **Parameters — вход; return — выход; overload — одно имя, разные формы входа.**
