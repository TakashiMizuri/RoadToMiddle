# Lambda expressions — краткая выжимка

> `1.3.6` · ~8 мин

## Главная идея

**Lambda** — compact syntax для anonymous function, конвертируется в **delegate** или **Expression tree**. **Closure** захватывает outer variables → display class → может продлить lifetime (events leak).

## Ключевые пункты

- Expression: `x => x * 2` — implicit return.
- Statement: `x => { ...; return v; }`.
- **`static` lambda** — no capture (C# 9+).
- **for + lambda** — все видят final `i`; fix: `int copy = i`.
- **foreach** — per-iteration variable (C# 5+).
- **async** → `Func<Task>`, not `Action`.
- EF: **`Expression<Func<T,bool>>`** для SQL translation.

## Примеры

```csharp
var evens = nums.Where(x => x % 2 == 0);
Func<Task> job = async () => await Work();
Func<int,int,int> add = static (a,b) => a+b;
```

## Частые ошибки

- Loop closure в `for`.
- Capturing `this` в long-lived delegate.
- `-=` с новой lambda.
