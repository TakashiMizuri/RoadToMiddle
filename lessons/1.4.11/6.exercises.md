# Exercises: CancellationToken

## Easy

### E1. Timeout Wrapper
**Goal:** Extension `WithTimeout(this Task task, TimeSpan)` using linked CTS.
**Success criteria:** Cancels on timeout; disposes CTS.

### E2. Register Cleanup
**Goal:** Async method opens `FileStream`; register close on token; trigger cancel mid-read.
**Success criteria:** Stream closed without leak.

## Medium

### M1. Linked Abort
**Goal:** Minimal API endpoint + `HttpClient` call; cancel when client disconnects (abort token).
**Success criteria:** Outbound request stops on tab close.

### M2. Loop Cancellation
**Goal:** `await foreach` over 1000 items; cancel after 10; measure how many processed.
**Success criteria:** Stops cooperatively between iterations.

## Hard

### H1. WhenAll + Token
**Goal:** Three parallel HTTP calls with shared linked token (user + 5s timeout); one slow URL — all cancel.
**Success criteria:** Token passed to each child; ties to `1.4.12`.
