# ArrayPool — краткая выжимка

> `1.5.14` · ~7 мин

## Главная идея

**ArrayPool** снимает временные **`T[]`** с GC через **Rent/Return**. Returned array часто **длиннее** запроса — используй logical length.

## Ключевые пункты

- **`ArrayPool<byte>.Shared.Rent(n)`** + **`Return`** в `finally`.
- **`clearArray: true`** — sensitive bytes.
- Не Return дважды; не use-after-return.
- Не класть rented array в поле класса.
- Large buffer → pool; tiny → stackalloc.

## Пример

```csharp
var buf = ArrayPool<byte>.Shared.Rent(512);
try { /* ... */ }
finally { ArrayPool<byte>.Shared.Return(buf); }
```

## Частые ошибки

- Забытый Return.
- Думать `buf.Length == 512`.
- Pool без clear в multi-tenant API.

## Мнемоника

> **Rent is a loan — Return in finally.**
