# Exercises: Assembly Metadata

## Easy
### E1. Assembly identity
**Goal:** Read manifest info.
**Steps:** `dotnet build` console app; print `Assembly.GetExecutingAssembly().FullName` and list referenced assemblies via reflection.
**Success criteria:** Name, version, at least 3 references listed.

## Medium
### M1. Namespace hunt
**Goal:** Namespace ≠ assembly.
**Steps:** Find which assembly defines `System.Text.Json.JsonSerializer` (docs or reflection). Compare namespace vs assembly name.
**Success criteria:** Correct assembly name (System.Text.Json).
