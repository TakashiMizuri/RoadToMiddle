# Exercises: Delegates — Multicast

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Multicast Logger
**Goal:** `LogHandler` delegate; register console + debug output; fire once; verify both outputs.
**Success criteria:** Order matches registration order.

### E2. Unsubscribe Bug
**Goal:** Reproduce failed `-=` with inline lambda; fix by storing delegate reference.
**Success criteria:** After unsubscribe, handler count via `GetInvocationList().Length` decreases.

## Medium

### M1. Resilient Dispatcher
**Goal:** `InvokeAll(Action<string> handlers, string msg)` — try/catch per handler, continue on failure.
**Success criteria:** Third handler runs even if second throws.

### M2. Return Value Trap
**Goal:** Multicast `Func<int>` with 3 handlers; demonstrate last-wins; rewrite to sum via invocation list.
**Success criteria:** Document behavior in comment.

## Hard (stretch)

### H1. Subscription Guard
**Goal:** Wrapper preventing duplicate `(target, method)` registration.
**Success criteria:** Same handler `+=` twice only adds once.
