## Why This Matters

Strings везде: HTTP, JSON, SQL parameters, logs, UI. **`string` в C# — не mutable char array**, а immutable UTF-16 sequence на heap. `s[0] = 'x'` не компилируется; `s += "!"` создаёт **новый** object. Immutability → allocations в loops и thread-safe reads без locks.

Backend: paths, queries, error messages. **`$"User {id}"`** readable, но компилируется в Format/handler machinery. Verbatim — regex и JSON. **Interning** dedupe literals — memory profiling и сюрпризы **`ReferenceEquals`**.

Связь с security (SQL injection), culture, GC, EF parameters.

## Core Concepts

### Reference type, immutable

```csharp
string name = "Ada";
// name[0] = 'B'; // error
```

**`System.String`** — sealed class. «Изменения» возвращают **новый** instance:

```csharp
string s = "hello";
string upper = s.ToUpperInvariant(); // s unchanged
```

Assignment копирует **reference**:

```csharp
string a = "hi";
string b = a;
```

### UTF-16 и `char`

**`char`** — 16-bit UTF-16 **code unit**. Emoji часто **surrogate pair** → **`Length` 2**:

```csharp
string emoji = "😀";
```

**`Length`** — code units, не graphemes. Grapheme-aware: **`Rune`**, **`StringInfo`**.

### Literal forms

```csharp
string path = "C:\\Users";
string verbatim = @"C:\Users";
string json = @"{ ""name"": ""Ada"" }";

string html = """
    <p>Hello</p>
    """;
```

### Interpolation и formatting

```csharp
int id = 42;
decimal total = 19.99m;
string msg = $"Order {id} total {total:C}";
string alt = string.Format("Order {0} total {1:C}", id, total);
```

Logs/API wire: **`CultureInfo.InvariantCulture`**. UI: **`CurrentCulture`**.

### Concatenation

```csharp
string full = first + " " + last;
```

Compile-time `"a" + "b"` → один literal. Runtime **`s += x` в loop** — O(n²) allocations → **`StringBuilder`** или **`string.Join`**.

### String interning

**Intern pool** — unique contents per process (conceptually). **Literals** с одинаковым текстом часто **один object**:

```csharp
string a = "hello";
string b = "hello";
ReferenceEquals(a, b); // often true
```

Runtime-built (`Concat`, `Substring`) — **обычно не interned**:

```csharp
string x = string.Concat("hel", "lo");
ReferenceEquals(a, x); // false
```

**`string.Intern(s)`** — canonical instance; **`IsInterned`**. Не intern unbounded user input — pool не освобождается.

### Null и empty

**`string?`**, **`string.Empty`** / **`""`**. Validation: **`IsNullOrWhiteSpace`**.

## Under the Hood

### Heap layout

String: header + length + immutable char data. Hash code **кэшируется** после первого **`GetHashCode()`** — важно для dictionary keys.

### `s += "x"`

1. Allocate buffer length+1
2. Copy + append
3. New reference
4. Old string — GC

### Interpolation lowering

Roslyn → **`string.Format`**, **`Concat`**, или **`DefaultInterpolatedStringHandler`** (.NET 6+). Const interpolation → compile-time literal.

### Intern pool

CLR hash table; literals из metadata **#Strings heap**. **`Intern(userInput)`** на unbounded input — memory leak risk.

using System.Globalization;
string log = string.Create(CultureInfo.InvariantCulture, $"Amount={amount}");

## Common Mistakes

- Ожидать mutation
- **`+` в tight loop**
- SQL через string concat — injection
- **`ReferenceEquals`** вместо value equality
- Intern user strings

## Production & Real-World Notes

Structured logging — objects, не always pre-interpolated. JSON UTF-8 ↔ UTF-16 internal. **`string.Create`**, **`Span<char>`** для high-throughput.
