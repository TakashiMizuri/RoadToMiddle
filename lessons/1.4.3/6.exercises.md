# Exercises: Task Class

## Easy
### E1. Status inspection
Create tasks: `FromResult`, `FromException`, canceled via token. Log Status and flags after completion.

## Medium
### M1. Observation experiment
Fire-and-forget faulting task; register `UnobservedTaskException`; observe behavior .NET 8.

### M2. WhenAll faults
Two tasks throw different exceptions; compare `await WhenAll` vs manual inspect.

## Hard
### H1. Safe fire-and-forget wrapper
Design helper that logs faults (concept only — no full production impl required).
