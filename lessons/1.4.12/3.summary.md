# Task.WhenAll / WhenAny — краткая выжимка

> `1.4.12` · ~8 мин

## Главная идея

Независимые async I/O — **запусти все Task'и, затем `await Task.WhenAll`**. Время ≈ **max**, не sum. **`WhenAny`** — первый завершившийся; отмени остальные.

## Ключевые пункты

- `var t1 = Foo(); var t2 = Bar(); await WhenAll(t1,t2);` — старт до WhenAll.
- Один fault → WhenAll fault.
- Token в каждый child (`1.4.11`).
- Не `.Result` до complete.
- CPU — throttle, не raw WhenAll на Task.Run (`1.4.10`).
