# Exercises: IAsyncDisposable

> Complete after passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Syntax Choice
**Goal:** For 3 types (List<int>, FileStream useAsync, custom IAsyncDisposable), pick `using` or `await using`.
**Success criteria:** 3/3 with reason.

### E2. Scope Skeleton
**Goal:** Write 8-line async method with `CreateAsyncScope` + resolve + SaveChanges.
**Success criteria:** `await using` on scope.

## Medium

### M1. Implement IAsyncDisposable
**Goal:** Class holding `Stream` — `DisposeAsync` disposes stream idempotently.
**Success criteria:** Guard flag; await if needed.

### M2. Anti-pattern Fix
**Goal:** Refactor class that calls `DisposeAsync().GetResult()` in `Dispose` — describe fix.
**Success criteria:** IAsyncDisposable-only or documented sync limitation.

## Hard (stretch)

### H1. Nested await using
**Goal:** `await foreach` + per-item `await using` lease — document dispose order on break/exception.
**Success criteria:** Inner-first disposal.
