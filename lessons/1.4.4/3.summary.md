# Task.Run — краткая выжимка

> `1.4.4` · ~10 мин

## Главная идея

**`Task.Run`** — очередь **CPU-bound** delegate в pool. **Не** для I/O. **`Task.Run(() => ReadAllText())`** — fake async, worker blocked.

## Ключевые пункты

- `await Task.Run` освобождает caller до старта/между await в lambda; CPU всё равно ест worker.
- Prefer **`Task.Run`** over **`StartNew`**.
- Bound concurrency: SemaphoreSlim, MaxDegreeOfParallelism.
- Micro work (<1ms) — sync inline.

## Примеры

```csharp
await Task.Run(() => BCrypt.HashPassword(p)); // ✓ CPU
await File.ReadAllTextAsync(path);            // ✓ I/O
```
