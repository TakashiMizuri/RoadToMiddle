# Answers: const vs readonly vs static readonly

## Scoring

- 90–100%: отлично
- 80–89%: pass
- <80%: повтори lection + Under the Hood

## Answers

### 1. [Recall]
**Answer:** Numeric primitives, **`bool`**, **`char`**, **`string`**, **`null`** reference — constant expressions only.
**Why:** ECMA constant field rules.

### 2. [Recall]
**Answer:** **Нет** — **`const`** implicitly static.
**Why:** Compile-time constant tied to type metadata, not instance.

### 3. [Understanding]
**Answer:** Compiler **inlined literal 10** в IL consumer; замена library DLL не меняет уже скомпилированные **`ldc.i4 10`** в consumer.
**Why:** Cross-assembly const embedding.

### 4. [Understanding]
**Answer:** Consumer loads field (**`ldsfld`**); deploy новой DLL с **`X = 40`** меняет behavior без recompile consumer (typical deploy scenario).
**Why:** No stale literal in consumer IL.

### 5. [Understanding]
**Answer:** Field initializer или любой constructor того же class/struct (once, before use completes).
**Why:** Definite assignment rules for readonly.

### 6. [Application]
**Answer:** **`static readonly`** (или static property with lazy init). **`const`** illegal; instance **`readonly`** — per object, не process-wide.
**Why:** Runtime **`UtcNow`**, single shared value.

### 7. [Application]
**Answer:** **`private readonly IEmailSender _sender`** set in constructor.
**Why:** Per-instance frozen dependency.

### 8. [Analysis]
**Answer:** **Нет** — not constant expression at compile time.
**Why:** Method call result unknown at compile time.

### 9. [Analysis]
**Answer:** **`const`** → often **`ldc.i4`** immediate in consumer; **`static readonly`** → **`ldsfld`**.
**Why:** Inlining vs field load.

### 10. [Understanding]
**Answer:** **Нет.** **`readonly struct`** ограничивает copying/mutation semantics struct; class fields отдельно.
**Why:** Different keywords, different features.

### 11. [Understanding]
**Answer:** Attribute args encoded in metadata at compile time — must be compile-time constants.
**Why:** Same const inlining/metadata pipeline.

### 12. [Application]
**Answer:** **`static readonly Regex`** — single shared instance, runtime creation.
**Why:** **`const`** cannot hold object instances from **`new`**.

### 13. [Analysis]
**Answer:** **`TypeInitializationException`** on every access to static members; type init failed permanently (until new AppDomain/process).
**Why:** Failed **`.cctor`** poisons type.

### 14. [Understanding]
**Answer:** **`IOptions`** reloadable/environment-specific; **`static readonly`** fixed at first type init.
**Why:** Configuration vs process-fixed constant.
