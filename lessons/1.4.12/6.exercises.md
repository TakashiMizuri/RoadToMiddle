# Exercises: Task.WhenAll / WhenAny

## Easy

### E1. Latency Compare
**Goal:** Two mock HTTP calls (200ms each) — measure sequential vs WhenAll with `Stopwatch`.
**Success criteria:** WhenAll ~200ms, sequential ~400ms.

### E2. Refactor Controller
**Goal:** Action with two sequential awaits → WhenAll; return combined JSON.
**Success criteria:** Both tasks started before await WhenAll.

## Medium

### M1. Partial Failure Log
**Goal:** WhenAll of three tasks where one throws — log all fault messages then rethrow.
**Success criteria:** Inspect each task in catch.

### M2. WhenAny Fallback
**Goal:** Call primary and backup API with WhenAny; cancel slower via linked CTS.
**Success criteria:** First success wins; loser canceled.

## Hard

### H1. Bounded Parallel Import
**Goal:** 100 URLs — WhenAll in batches of 10 with SemaphoreSlim + shared token.
**Success criteria:** No starvation; cancel stops batch.
