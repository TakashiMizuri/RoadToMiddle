# Примитивные типы — краткая выжимка

> `1.1.3` · ~10 мин на повторение

## Главная идея

Примитивы C# — **aliases** для CLR **value types** (structs). Assignment **копирует bits**. **`int`** — default integer; **`decimal`** — base-10 для денег; **`float`/`double`** — IEEE binary с ошибками округления.

## Ключевые пункты

- `int`→Int32, `long`→Int64, `double`→Double, `decimal`→Decimal, `bool`→Boolean, `char`→Char (UTF-16 unit).
- Literals: `42`, `42L`, `3.14` (double), `3.14F`, `19.99m`, `'A'`, `0xFF`, `0b1010`.
- **`decimal`**: suffix `m`, ~28–29 significant digits, медленнее double.
- **`double`/`float`**: не использовать для money; осторожно с `==`.
- **Overflow**: unchecked wrap; **`checked`** → `OverflowException`.
- **Boxing** `int` → `object` — heap allocation.

## Самые важные примеры

```csharp
decimal total = 100.00m * (1 - 0.15m); // 85.00
long safe = (long)a * b; // если product может > int.MaxValue
```

## «Под капотом» в одном абзаце

Value types в IL через load/store; JIT maps locals to registers/stack. `decimal` — software arithmetic на 128-bit struct. `float`/`double` — IEEE 754 binary; многие десятичные дроби не имеют exact binary representation. Boxing упаковывает value type в heap object для `object`/interface dispatch.

## Частые ошибки

- `double` для денег
- `0.1 + 0.2 == 0.3`
- Забытый `m`/`L`/`F`
- Multiply в `int` до cast в `long`

## Мнемоника

> **I**ndex/count → **int**; **L**arge ID/time → **long**; **M**oney → deci**m**al; **D**efault fraction literal → **double**.
