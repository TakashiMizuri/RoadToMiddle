## Answers

### 1. [Recall]
**Answer:** **C# 8** (.NET Core 3.0 / C# 8.0).
**Why:** Feature introduction version.

### 2. [Recall]
**Answer:** **Нет** — instance fields на interface не allowed (classic model).
**Why:** Interfaces не хранят per-object state.

### 3. [Understanding]
**Answer:** **Evolve interfaces** добавлением members **без break** existing implementers — default body даёт fallback behavior.
**Why:** Primary design motivation.

### 4. [Understanding]
**Answer:** **Extension** — static method, syntax `obj.Ext()` через static class; **DIM** — instance method on interface, metadata on interface, dispatch через interface mapping.
**Why:** Different mechanisms and call paths.

### 5. [Understanding]
**Answer:** **`C`** не declares public **`M`**; default accessible only via **`IA`** (interface reference). **`new C().M()`** — CS1061.
**Why:** Default not promoted to class public API.

### 6. [Application]
**Answer:** **Да**, compiles — inherits default **`Shutdown`**.
**Why:** Non-breaking addition.

### 7. [Application]
**Answer:** **`public void Shutdown() { ... }`** on class (implicit override) или explicit **`void IPlugin.Shutdown()`**.
**Why:** Normal interface implementation rules.

### 8. [Analysis]
**Answer:** **Compile error** — ambiguous/conflicting defaults; class must provide **`Log`** or explicit per-interface impl.
**Why:** Diamond resolution required.

### 9. [Analysis]
**Answer:** **Override** — hex formatting from class.
**Why:** Class member beats interface default.

### 10. [Understanding]
**Answer:** Когда subtypes нужны **shared instance fields, constructors, template method state** — abstract class (`1.2.8`).
**Why:** DIM cannot carry instance state.

### 11. [Application]
**Answer:** Cast/reference as **`IRequest`** (или generic constraint on interface) и вызов **`Validate()`** — default runs if not overridden.
**Why:** Default invoked through interface dispatch.

### 12. [Understanding]
**Answer:** **Class public (or explicit) implementation** overrides; default used only if no class implementation in mapping.
**Why:** Most specific wins.
