# LINQ SelectMany — краткая выжимка

> `1.3.22` · ~8 мин

## Главная идея

**`SelectMany`** проецирует каждый outer element в **inner sequence** и **склеивает** в один flat **`IEnumerable`**.

## Ключевые пункты

- **`Select`** → может дать **`IEnumerable<IEnumerable<T>>`**; **`SelectMany`** → **`IEnumerable<T>`**.
- Overload 2: `(outer, inner) => result` без отдельного Select.
- Left join: **`GroupJoin` + SelectMany(..., DefaultIfEmpty())`**.
- Query: несколько **`from`** без join → **`SelectMany`**.
- **`null`** inner → exception; use **`Empty`**.
- **`SelectMany(a => allBs)`** — cross product, не join по ключу.

## Примеры

```csharp
orders.SelectMany(o => o.Lines);
orders.SelectMany(o => o.Lines, (o, l) => new { o.Id, l.Qty });
```

## «Под капотом»

Lazy concat enumerators; total work O(output count). EF → APPLY/join expansion.
