## Answers

### 1. [Recall]
**Answer:** **`ReturnType IInterfaceName.MemberName(params) { }`** — без **`public`** перед типом.
**Why:** Syntax requirement for explicit.

### 2. [Recall]
**Answer:** **Нет** — CS1061; member not on class public API.
**Why:** Explicit not exposed on class type.

### 3. [Understanding]
**Answer:** Implicit — **public** на class, виден в IntelliSense и через interface. Explicit — **не public** на class, только через interface-typed reference.
**Why:** Access rules differ by design.

### 4. [Understanding]
**Answer:** **`CompareTo`** — implementation detail для sorting; не часть domain public API **`Customer`**.
**Why:** Hide auxiliary contract member.

### 5. [Understanding]
**Answer:** **`void IA.Reset()`** и **`void IB.Reset()`** — explicit per interface (или один public + один explicit если semantics совместимы — редко).
**Why:** Disambiguate same signature.

### 6. [Application]
**Answer:** **Explicit** — **`void ILogger.Log(...)`**.
**Why:** Hide from **`QuietLogger`** public surface.

### 7. [Application]
**Answer:** **Implicit** — **`public Task ChargeAsync(...)`**.
**Why:** Interface is primary public contract.

### 8. [Analysis]
**Answer:** **Нет**, CS1061.
**Why:** Only explicit implementation exists.

### 9. [Analysis]
**Answer:** **`1`** — call через **`IMsg`** uses **explicit** mapping **`IMsg.Send`**, not public **`Send`** (when both exist, interface call uses explicit impl for that interface).
**Why:** Interface dispatch to mapped explicit method.

### 10. [Understanding]
**Answer:** **Нет** — compile error; explicit cannot be declared **`public`**.
**Why:** Language rule.

### 11. [Application]
**Answer:** **`string IReadOnlyById.this[int id] => ...`** — explicit indexer.
**Why:** No public indexer on **`Registry`**.

### 12. [Understanding]
**Answer:** **InterfaceImpl** entry maps **`IReadOnlyById.get_Item`** → private explicit method on class.
**Why:** CLR interface dispatch path.

### 13. [Analysis]
**Answer:** **Derived's explicit `IFoo.M`** when object runtime type is **`Derived`** and reference is **`IFoo`**.
**Why:** Most specific implementation in mapping for derived type.

### 14. [Application]
**Answer:** Ломает discoverability и normal DI usage — callers expect **`OrderService`** public methods, not casts для каждого call.
**Why:** Explicit — tool for collisions/hiding, not default encapsulation.
