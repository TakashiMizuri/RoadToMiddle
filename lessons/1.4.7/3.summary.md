# State machine / MoveNext — краткая выжимка

> `1.4.7` · ~10 мин

## Главная идея

**`IAsyncStateMachine.MoveNext()`** — реальное тело **`async` method**, разрезанное на **`switch(<>1__state)`** по каждому **await**. **`AwaitUnsafeOnCompleted`** регистрирует resume; **`SetResult`/`SetException`** завершают Task caller.

## Ключевые пункты

- **`<>1__state`** — номер segment (-1 start, 0..N resume, -2 done).
- **`<>t__builder`** — **AsyncTaskMethodBuilder**.
- **`<>u__1`** — **TaskAwaiter** fields.
- Struct SM может **box** при first suspend (**SetStateMachine**).
- Stack: **`+<Method>d__2.MoveNext()`** — норма.

## «Под капотом»

IL — branches вместо C# switch. Continuation = повторный **MoveNext** с saved state. Hung Task — never SetResult или deadlock.
