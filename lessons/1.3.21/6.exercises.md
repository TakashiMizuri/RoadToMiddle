# Exercises: LINQ Join and GroupJoin

> Complete after lectures and passing `4.test-yourself.md`.

## Easy

### E1. Inner join report
**Goal:** **`Join`** basics.

**Tasks:** Given `List<Customer>` and `List<Order>`, produce `{ CustomerName, OrderId, Total }` for all orders with valid customer.

**Success criteria:** Customers with no orders absent; orphan orders absent.

---

### E2. GroupJoin hierarchy
**Goal:** Nested shape.

**Tasks:** **`GroupJoin`** → `{ Name, OrderCount, TotalSum }` per customer (Sum 0 if no orders).

**Success criteria:** All customers present.

---

## Medium

### M1. Left join flat
**Goal:** DefaultIfEmpty pattern.

**Tasks:** Same data — flat rows `{ Name, OrderId?, Total? }` including customers without orders (nulls).

**Success criteria:** Count rows = customers + (orders - matched pairs adjustment) — each customer at least once.

---

### M2. Query syntax rewrite
**Goal:** Method ↔ query equivalence.

**Tasks:** Rewrite M1 in query syntax with **`join into`**.

**Success criteria:** Same rows as M1.

---

## Hard

### H1. Three-way join
**Goal:** Chain joins.

**Tasks:** Customers → Orders → OrderLines; project `{ Customer, OrderId, ProductName, Qty }` inner join chain.

**Success criteria:** Correct filtering; no manual nested foreach.
