### 10. [Analysis]
Какое утверждение **неверно**?

- A) `var` и explicit `int` для `var x = 1` дают одинаковый IL.
- B) `var` позволяет присвоить переменной значение другого типа в runtime, если оно «совместимо».
- C) Public method return type не может быть объявлен через `var`.
- D) Target-typed `new` (`List<int> xs = new();`) снижает необходимость дублировать тип.
