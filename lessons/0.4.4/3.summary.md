# UDP — краткая выжимка

> `0.4.4` · ~7 мин

## Главная идея

**UDP** — connectionless **datagrams** без ACK/retransmit. Выбирают, когда **latency важнее полноты** (video, games) или сообщение маленькое (DNS). HTTP API и DB — **TCP** (`0.4.3`).

## Ключевые пункты

- Datagram boundary vs TCP byte stream.
- DNS :53 UDP, large → TCP fallback.
- TCP HOL blocking вреден для realtime.
- QUIC/HTTP/3 — reliability over UDP (`0.3.13`).
- Firewall: UDP 53, UDP 443 для HTTP/3.

## Частые ошибки

Block all UDP; UDP для critical data без app-level reliability.
