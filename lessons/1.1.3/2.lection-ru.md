## Why This Matters

Каждое объявление переменной выбирает тип — range, precision, память, контракт API. Backend: количества (`int`), ticks (`long`), налоги (`decimal`), датчики (`double`), флаги (`bool`). `float` для валюты или `int` для миллиардов операций — production-баги под нагрузкой.

Primitives — **aliases** для CLR value types в `System`. **Value type** копирует bits при assignment — в отличие от **reference types** (`string` в `1.1.4`).

Middle должен отвечать в review: «Почему `decimal`?», «Переполнится ли `int`?», «Безопасно ли `double ==`?». Связь с boxing, PostgreSQL (`numeric` vs `double precision`).

## Core Concepts

### Keywords и CLR types

| C# | CLR | Размер | Примечание |
|----|-----|--------|------------|
| `bool` | Boolean | 1 byte | true/false |
| `int` | Int32 | 4 bytes | default integer |
| `long` | Int64 | 8 bytes | большие диапазоны |
| `char` | Char | 2 bytes | UTF-16 code unit |
| `float` | Single | 4 bytes | IEEE binary32 |
| `double` | Double | 8 bytes | IEEE binary64, default float literal |
| `decimal` | Decimal | 16 bytes | base-10, ~28–29 digits |

`int` — счётчики, индексы, ID. **`long`** — Unix ms, snowflake ID. **`decimal`** — деньги. **`double`** — инженерная математика.

### Value type semantics

Locals conceptually на **stack** (JIT может держать в register). Assignment **копирует**:

```csharp
int a = 10;
int b = a;
b = 20; // a == 10
```

Parameters — by value (пока не `ref`/`out`). Bare `int` **не nullable** — нужен `int?` (`1.1.18`).

### Integer literals

```csharp
int dec = 42;
int hex = 0x2A;
int binary = 0b101010;
long big = 1_000_000_000L;
uint u = 100U;
```

`_` — для читаемости. Literal без suffix — **`int`**; большие — **`L`**.

### `float` и `double`

**IEEE 754**. Literal `3.14` — **`double`**; `float` нужен **`F`**:

```csharp
double pi = 3.141592653589793;
float approx = 3.14F;
```

Binary float **не представляет все десятичные дроби точно** — `0.1 + 0.2 != 0.3`. Для денег — **`decimal`**, не `==` для `double`.

Special: `NaN`, `Infinity`.

### `decimal`

128-bit **decimal floating**, для финансов:

```csharp
decimal price = 19.99m;
decimal tax = price * 0.08m;
```

Suffix **`m`**. Медленнее `double`, но base-10 precision. Хорошо maps на PostgreSQL **`numeric`**.

### `bool` и `char`

**`bool`** — только `true`/`false`, без implicit conversion из чисел (в отличие от C).

**`char`** — один **UTF-16 code unit**:

```csharp
char letter = 'A';
char uni = '\u0061';
```

Emoji часто — surrogate pair (2× `char`) или `string`/`Rune`.

### Conversions и overflow

Implicit widening: `int` → `long`, `int` → `double`. Explicit narrowing: `(int)3.99` → `3`.

Default **unchecked** — overflow integer **wraps**. **`checked`** — **`OverflowException`**:

```csharp
checked
{
    int oops = int.MaxValue + 1;
}
```

## Under the Hood

### CLR storage

Primitives — **structs** в `System`. IL: `ldc.i4`, `conv.i8`, … JIT кладёт в registers/stack slots.

**`bool`** — 1 byte в memory. **`decimal`** — struct со scale, sign, 96-bit mantissa; arithmetic в software.

### IEEE 754

**`float`**: sign + exponent + mantissa. Ошибки округления — свойство binary representation, не «баг .NET».

### Boxing

Присвоение value type в **`object`** → **boxing** (heap + copy):

```csharp
object boxed = 42;
int x = (int)boxed;
```

Hot paths: **`List<int>`**, generics вместо `ArrayList`.

## Examples

### Деньги

```csharp
decimal subtotal = 100.00m;
decimal total = subtotal - subtotal * 0.15m;
Console.WriteLine(total); // 85.00
```

### Overflow

```csharp
int items = 2_000_000_000;
long safe = (long)items * 2; // cast ДО multiply если product > MaxValue
```

## Common Mistakes

- **`double` для currency**
- **`==` для float/double**
- Забыть **`m`** на decimal literal
- **`char` как «буква Unicode»** без surrogate awareness
- Multiply в `int` перед cast в `long` когда product уже overflow

## Production & Real-World Notes

JSON: numbers → often `double`; для money — `decimal` или string + invariant culture. EF/PostgreSQL: `numeric` ↔ `decimal`.

## Key Takeaways

- Keywords = CLR structs; **value semantics**, copy on assign.
- **`decimal`** — money; **`double`** — science; know IEEE limits.
- Suffixes **`L`**, **`F`**, **`M`** обязательны где нужно.
- **`checked`** vs unchecked overflow.
- **Boxing** — heap cost.
