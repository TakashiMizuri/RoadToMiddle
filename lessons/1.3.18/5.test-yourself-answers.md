## Answers

### 1. [Recall]
**Answer:** **Last element** of array (index `Length - 1`).
**Why:** `^1` = one from end.

### 2. [Recall]
**Answer:** **0, 1, 2** — not 3.
**Why:** End exclusive.

### 3. [Recall]
**Answer:** **`^0`** would mean «zero from end» past last — **undefined**; API throws.
**Why:** Spec excludes `^0`.

### 4. [Understanding]
**Answer:** **Exclusive end** — `[0..3)` includes 0,1,2; length 3.
**Why:** Half-open interval.

### 5. [Understanding]
**Answer:** **String** creates **new string** (heap copy). **Span** is **view** over existing memory — no copy until `ToString()`.
**Why:** String immutability.

### 6. [Application]
**Answer:** `{ 40, 50 }` — from index Length-2 to end.
**Why:** `^2..` starts at index 3 (0-based) for length 5.

### 7. [Application]
**Answer:** `{ 2 }` — только index 1; end `^1` = exclusive boundary at index 2, поэтому `[1..2)` даёт один элемент.
**Why:** На `{1,2,3}` range `[1..^1)` эквивалентен `[1..2)`.

### 8. [Application]
**Answer:** `"de"` — indices 3 и 4 в `"abcdef"`.
**Why:** Start `^3` → index 3 (`'d'`); end `^1` exclusive → до index 5 не включая, т.е. 3 и 4.

### 9. [Application]
**Answer:** **New array copy** of entire sequence (same elements, new array instance).
**Why:** Range on array allocates copy.

### 10. [Analysis]
**Answer:** **Equivalent** for `line.Length > 0`. Both exclude last char. Empty string: both yield empty.
**Why:** `..^1` is idiomatic same as `[0..Length-1]`.

### 11. [Analysis]
**Answer:** **New List segment copy** (internal implementation), **not Span** by default.
**Why:** List<T> range indexer returns copied sub-list.

### 12. [Understanding]
**Answer:** `{ 1, 2, 3 }` — indices 1, 2, 3; исключены index 0 и index 4.
**Why:** Start 1 inclusive; end `^1` exclusive at last index (4).

### 13. [Application]
**Answer:** **No** — throws (empty string, ^1 invalid or out of range).
**Why:** Must check Length >= 1.

### 14. [Recall]
**Answer:** C# uses **`^` prefix** from end; **`^1` is last**; no negative literal index like Python `-1` in syntax.
**Why:** Distinct Index type model.

### 15. [Application]
**Answer:** `s[^3..]` or `s[(s.Length - 3)..]`
**Why:** Last three chars from index Length-3.
