# Exercises: TimeProvider (.NET 8+)

> Complete after reading `1.lection-eng.md`, `2.lection-ru.md` and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Injectable "now"
**Goal:** Replace `DateTime.UtcNow` with `TimeProvider.GetUtcNow()`.
**Steps:** Create `net8.0` class library with `LicenseChecker` that returns true if `GetUtcNow() < expiry`. Write xUnit test with `FakeTimeProvider` and `SetUtcNow` past expiry.
**Success criteria:** Test passes without `Thread.Sleep`; production code has zero `DateTime.UtcNow`.

### E2. DI registration
**Goal:** Wire `TimeProvider` in minimal host.
**Steps:** `WebApplication.CreateBuilder`, register a singleton service using `TimeProvider`, resolve in test with `FakeTimeProvider` override.
**Success criteria:** Integration test proves override works.

## Medium

### M1. Retry backoff under fake time
**Goal:** Test exponential backoff timing.
**Steps:** `ApiClient` with 3 retries, delays 100ms → 200ms → 400ms via `timeProvider.Delay`. Mock HTTP to fail twice then succeed. Use `FakeTimeProvider.Advance` between assertions.
**Success criteria:** Exactly 3 HTTP calls; total test time < 2 seconds.

### M2. Timeout wrapper
**Goal:** Build `WithTimeoutAsync(Func<Task>, TimeSpan, TimeProvider, CancellationToken)`.
**Steps:** Race operation task against `timeProvider.Delay(timeout)`. On timeout throw `TimeoutException`.
**Success criteria:** Test: slow operation cancelled when `Advance(timeout)`; fast operation completes before advance.

## Hard (stretch)

### H1. Polly integration
**Goal:** Configure Polly v8 retry with `ResiliencePipelineBuilder` time provider (or custom delay in handler using injected `TimeProvider`).
**Steps:** Document how delays flow through your abstraction; test 5 failures with jitter disabled.
**Success criteria:** Deterministic test with `FakeTimeProvider`.

## Submission checklist

- [ ] No `Task.Delay` in code under test
- [ ] `FakeTimeProvider` isolated per test
- [ ] `dotnet test` green in < 5s for timing tests
