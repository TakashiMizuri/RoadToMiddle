# Exercises: Accessible HTML Tables

> Complete after lectures and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Fix the header row
**Goal:** Convert presentational cells to semantic headers.

Given:

```html
<table>
  <tr>
    <td><strong>Product</strong></td>
    <td><strong>Stock</strong></td>
  </tr>
  <tr>
    <td>Widget</td>
    <td>42</td>
  </tr>
</table>
```

Rewrite with `<thead>`, `<tbody>`, `<th scope="col">`, and `<caption>`.

**Success criteria:** Valid structure; axe/Lighthouse no "table header" violation; caption describes inventory.

---

### E2. Add row headers
**Goal:** Apply `scope="row"` for the first column.

Extend E1 so rows are regions: North, South, East with stock counts in one "Units" column (two columns total: Region, Units).

**Success criteria:** First column uses `<th scope="row">`; second column `<td>`.

---

## Medium

### M1. Build a comparison table
**Goal:** Hands-on markup for a pricing matrix.

Create an HTML file with a table comparing three plans (Free, Pro, Team) and features (Storage, Support, SSO) as checkmarks or Yes/No text — **not color alone**.

Include caption, thead, tbody, appropriate scopes.

Validate with [W3C Nu Html Checker](https://validator.w3.org/nu/) and browser DevTools Accessibility tree.

**Success criteria:** 3×3+ grid; both col and row headers where needed; passes validator; AT tree shows columnheader/rowheader.

---

### M2. Spot the layout table
**Goal:** Refactor anti-pattern.

```html
<table width="100%">
  <tr>
    <td><nav>...</nav></td>
    <td><main>...</main></td>
    <td><aside>...</aside></td>
  </tr>
</table>
```

Sketch semantic replacement using `<header>`, `<nav>`, `<main>`, `<aside>` and CSS Grid. List two accessibility improvements.

**Success criteria:** No table for layout; landmarks identified; explains AT announcement fix.

---

## Hard (stretch)

### H1. Complex headers sketch
**Goal:** Plan `headers`/`id` for a quarter-over-year table.

Draw on paper or mermaid: rows = regions, columns = Q1–Q4 grouped under 2025 and 2026. Write **one** example row of HTML showing `id` on merged year header and `headers` on one `<td>`.

**Success criteria:** Unique ids; td headers attribute lists all applicable header ids; explains why scope alone fails here.
