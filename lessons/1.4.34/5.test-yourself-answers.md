# Answers: BackgroundTaskQueue

## Scoring

- 90–100%: отлично · 80–89%: pass · <80%: повтори lection + summary

## Answers

### 1. [Recall]
**Answer:** Безопасно отложить work после HTTP response с scope и exception handling.
**Why:** Replaces fire-and-forget.

### 2. [Recall]
**Answer:** Один буфер на приложение; stateless; consumer singleton hosted.
**Why:** Shared queue across requests.

### 3. [Understanding]
**Answer:** `IServiceScopeFactory.CreateAsyncScope()` на каждый dequeued item, resolve `DbContext` inside.
**Why:** Per-job scoped lifetime.

### 4. [Understanding]
**Answer:** Producer await при полной очереди — backpressure, не unbounded memory.
**Why:** BoundedChannelFullMode.Wait.

### 5. [Application]
**Answer:** `orderId` (value); не `DbContext` (scoped, disposed).
**Why:** Closure lifetime vs request scope.

### 6. [Application]
**Answer:** В consumer `try/catch` around await work.
**Why:** Request already completed.

### 7. [Analysis]
**Answer:** Head-of-line blocking; latency растёт; нужны multiple consumers или faster jobs.
**Why:** Single-threaded consumer model.

### 8. [Understanding]
**Answer:** Channel — process memory, no survive crash; Rabbit — durable, retry, DLQ.
**Why:** Durability trade-off.

### 9. [Application]
**Answer:** `AddSingleton<IBackgroundTaskQueue, BackgroundTaskQueue>(); AddHostedService<QueuedHostedService>();`
**Why:** Standard registration.

### 10. [Analysis]
**Answer:** Channel consumer await observes; `_ =` → unobserved risk.
**Why:** Exception paths.

### 11. [Understanding]
**Answer:** Async dispose для `IAsyncDisposable` services (e.g. async cleanup).
**Why:** `1.4.35` integration.

### 12. [Recall]
**Answer:** `Channel<T>` bounded or unbounded.
**Why:** BCL producer-consumer.

### 13. [Application]
**Answer:** Capture `TraceIdentifier` in closure; `LogContext.PushProperty` inside work.
**Why:** Ambient logging across async work.

### 14. [Understanding]
**Answer:** Cancels `ReadAsync`/work; graceful shutdown policy.
**Why:** Host lifetime.

### 15. [Application]
**Answer:** POST awaits SaveChanges → enqueue delegate with orderId → 202; hosted loop dequeues, scope, await email send, log errors.
**Why:** End-to-end safe pattern.
