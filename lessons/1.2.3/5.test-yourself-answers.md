## Answers

### 1. [Recall]
**Answer:** Имя = class, **no return type**, вызывается только при **`new`**, не напрямую как method.
**Why:** Special instance initialization hook.

### 2. [Recall]
**Answer:** Когда class уже объявляет **любой instance constructor** explicitly — parameterless не генерируется автоматически.
**Why:** Compiler rule once custom ctor exists.

### 3. [Recall]
**Answer:** Делегирует другому constructor **того же class**; должен быть **first statement**; target ctor выполняется до завершения текущего.
**Why:** Constructor chaining on same type.

### 4. [Understanding]
**Answer:** **`: base()`** (если есть) → chained ctor (this/base target) → текущий body → затем **object initializer** assignments.
**Why:** Initialization order spec.

### 5. [Understanding]
**Answer:** **DRY** — одна validation/invariant gate; overloads только convenience.
**Why:** Maintain single source of truth.

### 6. [Application]
**Answer:** **Нет**, нужен parameterless ctor или аргумент.
**Why:** No implicit default after explicit ctor.

### 7. [Application]
**Answer:** **DI container** resolve `ILogger` и вызывает synthesized constructor с dependency.
**Why:** Primary ctor = constructor for injection.

### 8. [Application]
**Answer:** Valid pattern — single-parameter ctor forwards default age to two-parameter ctor with rules.

### 9. [Analysis]
**Answer:** **Нет** — только один constructor initializer (`: this` **или** `: base`).
**Why:** Language grammar.

### 10. [Analysis]
**Answer:** Base ctor мог partially initialize; throw оставляет unusable object; callers должны handle exception; иногда лучше factory/validation before construction.
**Common mistake:** Ignoring base ctor side effects.

### 11. [Understanding]
**Answer:** **`: base`** → **base class** constructor. **`: this`** → **another ctor same class**.
**Why:** Different delegation targets.

### 12. [Application]
**Answer:** **Named intent**, скрыть `new`, enforce invariants, optional caching/reuse, restrict public construction API.
**Why:** Factory pattern benefits.

### 13. [Understanding]
**Answer:** **Constructor** (или private setter/full property) — core invariant; initializer только если ctor уже гарантирует valid baseline.
**Why:** Cannot skip ctor path.

### 14. [Recall]

**Why:** Primary constructors feature version.
