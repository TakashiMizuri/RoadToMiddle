# Span, ReadOnlySpan, Memory — краткая выжимка

> `1.5.12` · ~10 мин · **глубоко**

## Главная идея

**Span/Memory** — typed **view** на contiguous memory **без копии**. **Span** — ref struct, sync-only; **Memory** — для async и полей класса.

## Ключевые пункты

- **`AsSpan()`**, slice `[start..end]`, bounds checked.
- **ReadOnlySpan** — parse/string scan без Substring.
- **ref struct**: no async, no boxing, stack discipline.
- **Memory** + **`WriteAsync`**, **`memory.Span`** sync.
- **stackalloc** → Span (`1.5.13`); **ArrayPool** (`1.5.14`).

## Примеры

```csharp
ReadOnlySpan<char> s = line.AsSpan();
int.TryParse(s, out var n);

ReadOnlyMemory<byte> m = buffer;
await stream.WriteAsync(m);
```

## «Под капотом»

Span хранит byref + length; не владеет памятью. GC pin при передаче в native.

## Частые ошибки

- Span из stackalloc наружу метода.
- Span в async method.
- Substring там, где достаточно Span до финального ToString.

## Мнемоника

> **Span for sync slices; Memory for await.**
