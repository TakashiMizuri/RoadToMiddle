## Answers

### 1. [Recall]
**Answer:** Scheme, host, port (часть authority), path, query, fragment.
**Why:** Классическая декомпозиция RFC 3986 / WHATWG.

### 2. [Recall]
**Answer:** 443.
**Why:** Default для https.

### 3. [Recall]
**Answer:** Fragment (`#...`).
**Why:** Не part of request target для origin HTTP.

### 4. [Understanding]
**Answer:** scheme=https; host=api.test.com; port=8443; path=/v2/items; query keys `tag` (два значения book, cd); fragment=details.
**Why:** Authority включает host:port; query до `#`.

### 5. [Understanding]
**Answer:** `https://shop.com/admin/api/export` — relative URL без leading `/` резолвится от directory текущего path.
**Why:** WHATWG relative resolution algorithm.

### 6. [Understanding]
**Answer:** Query логируется proxies, web server access logs, browser history, Referer; легко утекает.
**Why:** Headers like Authorization контролируются лучше и не в URL по convention.

### 7. [Understanding]
**Answer:** Path param идентифицирует конкретный resource в иерархии (`/users/42`); query — filter/lookup по коллекции (`id=42` как parameter).
**Why:** REST style предпочитает path для identity; query для optional scoping.

### 8. [Application]
**Answer:** `color=blue%20green` (или `color=blue%20green` через encodeURIComponent всей value).
**Why:** Space → `%20` в query component.

### 9. [Application]
**Answer:** `GET /search?q=http HTTP/1.1` (+ Host header отдельно). Value `http` в query должно быть encoded как `q=http` или `q=http%3A%2F%2F` если literal — в exam достаточно показать awareness encoding для `:`
**Why:** Request line = method + target + version.

### 10. [Analysis]
**Answer:** B неверно.
**Why:** Fragment не приходит на server в standard HTTP request parsing.

### 11. [Analysis]
**Answer:** После decode `%25` → `%`, value близко к `data%20set` или literal `%2520` depending on layers — типичная ошибка **double encoding**.
**Why:** Encode once at URL construction.

### 12. [Understanding]
**Answer:** Origin = scheme + host + port; hostname `localhost` ≠ `127.0.0.1` даже при same machine.
**Why:** CORS exact string match.

### 13. [Application]
**Answer:** `/api/v1`.
**Why:** Absolute path `/api/v1` replaces pathname when second arg is base with path.

### 14. [Recall]
**Answer:** `host[:port]` (+ optional userinfo), между `://` и path.
**Why:** Authority identifies origin server endpoint.

### 15. [Understanding]
**Answer:** Когда server слушает не default port — dev Docker, TLS termination on 8443, internal services.
**Why:** Client must connect to correct TCP port.
