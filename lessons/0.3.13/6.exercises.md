# Exercises: HTTP versions overview

> Conceptual + optional curl observation. Hints via AI.

## Easy

### E1. Protocol column safari
**Goal:** See versions in the wild.

**Steps:**
1. Open any major site → DevTools Network → enable Protocol column.
2. Reload; note mix of `h2`, `h3`, `http/1.1` if visible.

**Success criteria:** One sentence per protocol you observed.

---

### E2. curl ALPN
**Goal:** Read verbose negotiation.

**Steps:**
1. `curl.exe -v https://www.google.com 2>&1` (or cloudflare.com).
2. Find ALPN line and HTTP version on response line.

**Success criteria:** State accepted ALPN token.

---

## Medium

### M1. Draw the fallback chain
**Goal:** Solidify mental model.

**Tasks:** ASCII diagram: Client → (try h3) → (fallback h2) → (fallback h1) → Origin. Label where CDN vs browser decides.

**Success criteria:** Diagram fits on one screen; labels UDP vs TCP.

---

## Submission checklist

- [ ] Observed at least one h2 or h3 in browser or curl
- [ ] Can explain TCP HoL vs QUIC stream isolation in own words
