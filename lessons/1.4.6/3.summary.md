# async/await compiler — краткая выжимка

> `1.4.6` · ~10 мин

## Главная идея

Compiler превращает **`async` method** в **`IAsyncStateMachine`** с **`MoveNext()`** и **`AsyncTaskMethodBuilder`**. **`await`** — checkpoint: если Task не completed, save state, register continuation, **return** (thread свободен).

## Ключевые пункты

- Caller получает **Task** сразу.
- Каждый **await** — возможная suspension.
- Completed await — sync path без suspension.
- Locals across awaits → fields state machine.
- Stack traces показывают **MoveNext** — normal.

## Пример (conceptual)

```csharp
// await http.GetAsync(url);
// → GetAwaiter, if !IsCompleted → state=1, AwaitOnCompleted, return
```

## «Под капотом»

Continuation вызывает MoveNext на другом pool thread. Builder SetResult завершает Task для caller.
