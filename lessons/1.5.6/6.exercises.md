# Exercises: GC Roots and Mark Phase

## Easy
### E1. Root diagram
**Goal:** Draw reachability.
**Steps:** Diagram: static → A → B; local in Main → C; B→C. Which survive if local nulled but static remains?
**Success criteria:** A,B,C live via static path.

## Medium
### M1. Event leak fix
**Goal:** Fix publisher-subscriber leak pattern from lecture.
**Steps:** Write `IDisposable` on subscriber unsubscribing in Dispose.
**Success criteria:** `-=` in Dispose called.

## Hard
### H1. WeakReference cache
**Goal:** Optional cache semantics.
**Steps:** Sketch cache using `WeakReference<Bitmap>` — when entry disappears; trade-offs.
**Success criteria:** Not a strong root alone.
