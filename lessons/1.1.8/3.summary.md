# Операторы — краткая выжимка

> `1.1.8` · ~8 мин на повторение

## Главная идея

**Operators** превращают типы `1.1.3`–`1.1.6` в вычисления и условия. Критично: **integer division**, **short-circuit `&&`/`||`**, **`??` defaults**, различие **value vs reference `==`**.

## Ключевые пункты

- **`5 / 2` → `2`** (int); для дроби — `5.0 / 2` или `decimal`.
- **`0.075m`** — decimal literal; деньги без `double`.
- **`checked`** — overflow → exception; default unchecked — wraparound.
- **`==` на reference types** — identity; **`string`** — content equality.
- **`&&` / `||`** — short-circuit; безопасно: `user != null && user.Active`.
- **`??`** — left if not null else right; chains left-associative.
- **`??=`** — assign if null (`list ??= new()`).
- **Ternary `? :`** — expression для двух веток.

## Самые важные примеры

```csharp
double avg = sum / 3.0;
decimal tax = price * 0.08m;
if (x != null && x.Count > 0) { }
var name = input ?? "Guest";
list ??= new List<int>();
```

## Частые ошибки

- `sum / 3` для среднего как `double`.
- `double` equality без tolerance.
- `|` вместо `||` с возможным NRE на RHS.

## Мнемоника

> **Делишь int — теряешь дробь; coalesce null — бери запасной; AND с null — сначала проверь ссылку.**
