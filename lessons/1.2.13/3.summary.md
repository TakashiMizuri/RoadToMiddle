# Records — краткая выжимка

> `1.2.13` · ~8 мин на повторение

## Главная идея

**Record** — тип для **immutable data** с **value equality** (сравнение по полям, не по ссылке) и **`with`** для копий с изменениями; компилятор синтезирует `Equals`, `GetHashCode`, `ToString`.

## Ключевые пункты

- **`record R(int A, string B)`** — positional: constructor + `{ get; init; }` properties.
- **`a == b`** — true при равных данных (в отличие от default class).
- **`var n = old with { B = "x" }`** — non-destructive update; `old` не меняется.
- **`with` — shallow copy**; `List<>` внутри record — общая ссылка.
- **`record struct`** / **`readonly record struct`** — value type на stack (C# 10+).
- **DTO/commands/events** — records; **EF entities** с change tracking — обычно class.
- **Deconstruct:** `(int id, string name) = dto;`
- JSON deserializer матчит constructor parameters с property names.

## Самые важные примеры

```csharp
public record CreateProductCommand(string Name, decimal Price, int CategoryId);

var updated = customer with { Email = "new@mail.com" };

public record Cart(IReadOnlyList<CartLine> Lines)
{
    public Cart AddLine(CartLine line) =>
        this with { Lines = Lines.Append(line).ToList() };
}
```

## Частые ошибки

- `Dictionary`/`List` в record без immutable wrapper — shared mutation.
- Record для ORM entity с navigations.
- Ожидание deep equality для вложенных reference types.

## Мнемоника

> **Record = данные + равенство по значению + `with`; мутабельность — новая копия, не тот же объект.**
