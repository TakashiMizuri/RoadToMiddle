# LINQ Take, Skip, TakeWhile, SkipWhile — краткая выжимка

> `1.3.26` · ~6 мин

## Главная идея

**`Take`/`Skip`** — slicing по count (pagination); **`TakeWhile`/`SkipWhile`** — slicing по predicate с **early stop**. Prefix operators ≠ **`Where`**.

## Ключевые пункты

- Pagination: `OrderBy(...).Skip((page-1)*size).Take(size)`.
- **`TakeWhile`**: prefix пока predicate true; stop на first false, tail не читается.
- **`SkipWhile`**: skip пока true; с first false — **весь остаток** без фильтра.
- EF: `Skip/Take` → `OFFSET/FETCH`; большой offset дорог.
- Negative count → exception.

## Примеры

```csharp
query.OrderBy(x => x.Id).Skip(20).Take(10);
nums.TakeWhile(n => n % 2 == 0);
lines.SkipWhile(l => l.StartsWith("#"));
```

## Частые ошибки

- Pagination без stable `OrderBy`.
- `Where` вместо `TakeWhile` для prefix.
- Ожидать фильтрацию от `SkipWhile` после boundary.
