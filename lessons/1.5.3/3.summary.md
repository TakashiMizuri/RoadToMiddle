# Assembly: metadata и manifest — выжимка

> `1.5.3` · ~6 мин

## Главная идея

**Assembly** — deployment unit CLR: **IL + metadata + manifest** (+ resources). **Metadata** — type database; **manifest** — identity и dependencies.

## Ключевые пункты

- **Namespace ≠ assembly**.
- Metadata tables: TypeDef, MethodDef, tokens в IL.
- Reflection читает те же tables, что JIT.
- Version mismatch → load failures.
