# Heap allocation и object headers — выжимка

> `1.5.5` · ~7 мин

## Главная идея

Reference objects на heap = **header (8) + MethodTable ptr (8) + fields + padding**. Min size ~24 bytes на 64-bit. Allocation cheap (Gen0 bump) до частых GC.

## Ключевые пункты

- MethodTable — type identity, vtable, GC scan map.
- Boxing/`new`/strings → heap pressure.
- Allocation rate → Gen0 frequency (`1.5.7`).
- ≥85KB → LOH (`1.5.8`).
