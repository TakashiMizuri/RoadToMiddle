# Exercises: AsyncLocal

> Complete after passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Middleware Sketch
**Goal:** Pseudocode middleware setting and clearing `AsyncLocal` correlation ID.
**Success criteria:** `try/finally` with null assign.

### E2. ThreadLocal vs AsyncLocal
**Goal:** Explain in 3 sentences why `ThreadLocal` breaks after `await Task.Delay(1)` in a request.
**Success criteria:** Mentions thread switch.

## Medium

### M1. Tenant Scope
**Goal:** Implement `IDisposable` helper `TenantScope.Use(guid)` wrapping `AsyncLocal<Guid>`.
**Success criteria:** Nested `using` restores or clears correctly.

### M2. Test Isolation
**Goal:** Two parallel async tests set different AsyncLocal values — assert no cross-talk (design only).
**Success criteria:** Each test clears in finally.
