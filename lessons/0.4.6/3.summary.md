# localhost, 127.0.0.1, 0.0.0.0 — краткая выжимка

> `0.4.6` · ~7 мин

## Главная идея

**Loopback** (`127.0.0.1`, `::1`) — traffic остаётся на host. **`0.0.0.0`** в listen = **all interfaces**, не URL для browser. **Bind** определяет, кто может подключиться по TCP (`0.4.3`).

## Ключевые пункты

- `localhost` → hosts/DNS → usually loopback (`0.4.5`).
- Bind `127.0.0.1` — только local; `0.0.0.0` — LAN + loopback.
- Client connect: конкретный IP/hostname, never `0.0.0.0`.
- Kestrel default localhost — phone/LAN need `ListenAnyIP`.
- Docker: loopback-only inside container breaks `-p` publish.

## Примеры

```
http://localhost:5000     → bind loopback OK locally
http://192.168.1.50:5000  → need 0.0.0.0 or NIC bind
```

## Частые ошибки

Открыть `http://0.0.0.0:8080` в browser; IPv6 localhost vs IPv4-only bind.
