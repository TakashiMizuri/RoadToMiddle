## Answers

### 1. [Recall]
**Answer:** Внутри method, constructor, property accessor или другой local function.
**Why:** Local function — nested function, не member типа.

### 2. [Recall]
**Answer:** **Нет** — только из enclosing member, где объявлена.
**Why:** Scope ограничен outer member.

### 3. [Understanding]
**Answer:** **Локальность** — helper рядом с использованием; не раздувает private API класса; читатель видит workflow целиком.
**Why:** Intent «существует только здесь».

### 4. [Understanding]
**Answer:** Запрещает capture outer locals/`this`; явный data flow через parameters; нет closure allocation.
**Why:** Static local = nested function без closure class.

### 5. [Understanding]
**Answer:** Local function — именованная nested function с **`ref`/`out`**, прямой рекурсией, часто без delegate; lambda — **delegate instance** для API вроде LINQ.
**Why:** Разные lowering сценарии и синтаксис.

### 6. [Application]
**Answer:** **`private static`** method (или validator class) — reuse в двух endpoints.
**Why:** Local function не видна из второго handler.

### 7. [Application]
**Answer:** **Local function** с `ref` — lambda в `Func<>` не поддерживает `ref`/`out` в том же виде.
**Why:** Ограничения delegate signatures.

### 8. [Analysis]
**Answer:** **Closure class** с полем для captured `order`; local function — method на closure.
**Why:** Та же модель, что у capturing lambda.

### 9. [Analysis]
**Answer:** Нет изолированных тестов, сложный review, смешение transport и domain; rule должна быть в service/testable static helper.
**Why:** Local functions — readability tool, не architecture layer.

### 10. [Understanding]
**Answer:** **Нет** — local functions не участвую in inheritance; модификаторы доступа недоступны.
**Why:** Не являются members типа в обычном смысле.

### 11. [Application]
**Answer:** **Две** — по state machine на каждый `async` method (outer и inner).
**Why:** Async local function компилируется отдельно.

### 12. [Recall]
**Answer:** **Да.**
**Why:** Полноценная function signature, в отличие от ограничений некоторых delegate forms.
