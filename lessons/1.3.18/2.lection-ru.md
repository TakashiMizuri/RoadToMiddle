# Index и Range (^1, ..)

> Roadmap: `1.3.18` · Узел: `1.3` — C#: продвинутые темы · Глубина: практика

## Learning Objectives

После этого урока ты сможешь:

- Использовать **`Index`** с **`^n`** (с конца) и **`0`** (с начала).
- Применять **`Range`** с **`start..end`** для slice arrays, spans, strings.
- Предсказывать **inclusive/exclusive** границы и invalidity **`^0`**.
- Slice без ручной арифметики `Length - 1`.
- Знать, какие типы поддерживают **Index/Range**.
- Избегать лишних allocations через **`Span<T>`** / **`ReadOnlySpan<char>`**.

---

## Why This Matters

Раньше «последний элемент» = `arr[arr.Length - 1]`, «всё кроме последнего» = `Substring(0, s.Length - 1)` — off-by-one и шум. **`^1`** читается как «last»; **`..`** — «slice from A to B».

Parsing stack traces, URL trim, last N log lines, tests на tail массива — ежедневные кейсы. С **Span** slicing может быть **allocation-free** на hot paths.

Middle dev использует index/range вместо length math — но помнит: **end exclusive**, **`^0` illegal**.

## Core Concepts

### Index: с начала или с конца

```csharp
int[] nums = { 10, 20, 30, 40 };
nums[0];    // 10
nums[^1];   // 40 — last
nums[^2];   // 30
```

**`^n`** — n elements from end; **`^1` = last**. **`^0` invalid** → `ArgumentOutOfRangeException`.

`new Index(2, fromEnd: true)` ≡ `^2`.

### Range: slices

```csharp
var slice = nums[1..3];   // index 1, 2 → { 20, 30 }
var tail = nums[2..];     // { 30, 40 }
var head = nums[..2];     // { 10, 20 }
var all = nums[..];
var lastTwo = nums[^2..];
```

**End exclusive:** `[0..3]` = indices 0,1,2. Half-open interval как в many APIs.

### Поддерживаемые типы

- **Arrays** `T[]`
- **String** — **new string** copy
- **Span<T>**, **ReadOnlySpan<T>**
- Types с **Length/Count** + indexer

**`List<T>`** — Index ok; range даёт **copy** slice. Zero-copy — array или Span.

### String examples

```csharp
string file = "report.pdf";
var ext = file[^3..];     // проверь length
var name = file[..^4];
```

Validate length перед `^n` на коротких строках.

### Index и Range как variables

```csharp
Range middle = 1..^1; // без first и last
var part = nums[middle];
```

## Under the Hood

`Index.GetOffset(length)` — bounds check. Range → start/end offsets; length = end - start.

`arr[^1]` → helper с `Index`. **String range** — O(n) copy. **Span range** — view без heap allocation.

---

## Examples

```csharp
int[] inner = data[1..^1]; // без первого и последнего

string ip = "192.168.0.1";
string lastOctet = ip[(ip.LastIndexOf('.') + 1)..];
```

## Common Mistakes

- **`^0`** — always throws.
- End **inclusive** misconception.
- **`^5`** on length-3 array.
- Range на **IEnumerable** без materialize.
- Python-style **negative index** — только `^n`.

## Production & Real-World Notes

Parsers: Span + IndexOf + ranges. Tests: `actual[^1]`. Logging tail — watch allocations на huge strings.

## Quick Reference

```
^1 last, ^0 INVALID
start..end — end EXCLUSIVE
..end | start.. | ..
Span — zero-copy
```

---

## Key Takeaways

- **`^1`** last; **`^0`** invalid.
- **`..`** end exclusive.
- String slice allocates; Span — нет.
- Validate length на коротких sequences.
