# Exercises: Async in ctor and property getters

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Compile-time traps
**Goal:** See compiler errors.
**Steps:** Try `async` ctor and `async` property in scratch project.
**Success criteria:** Record exact error messages.

### E2. CreateAsync factory
**Goal:** Replace blocking ctor init.
**Steps:** Refactor service that used `.Result` in ctor to `CreateAsync` + await at `Main`.
**Success criteria:** No sync block; tests green.

## Medium

### M1. IHostedService warmup
**Goal:** ASP.NET cache preload.
**Steps:** Minimal API + singleton cache + `IHostedService.StartAsync` loads data.
**Success criteria:** First request never sees empty cache.

### M2. Interview answer doc
**Goal:** Write 150-word answer: "Why no async properties?"
**Success criteria:** Mentions sync contract, hidden suspension, async methods alternative.

## Hard (stretch)

### H1. IAsyncInitializable pipeline
**Goal:** Enforce init order at startup.
**Steps:** Register multiple services; extension method `await app.InitializeAsync()` before `Run`.
**Success criteria:** Throws if `UseBeforeInit` called early.
