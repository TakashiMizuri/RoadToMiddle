# Func, Action, Predicate — краткая выжимка

> `1.3.4` · ~8 мин

## Главная идея

**`Action`** — void callback; **`Func<..., TResult>`** — return value (последний type param); **`Predicate<T>`** = **`Func<T, bool>`**. Стандарт BCL вместо custom delegates.

## Ключевые пункты

- `Action`, `Action<T>`, … до 16 params.
- `Func<TResult>`, `Func<T, TResult>`, … — return всегда последний.
- LINQ: `Where` → `Func<T,bool>`, `Select` → `Func<T,TResult>`.
- Method group: `Func<int,string> f = int.ToString;`
- Variance: `Func<in T, out R>` как у delegates (`1.3.2`).
- **`async void`** с `Action` — anti-pattern; **`Func<Task>`**.

## Примеры

```csharp
await Task.Run(() => Compute());
var evens = nums.Where(n => n % 2 == 0);
List<string> a = names.FindAll(s => s.StartsWith("A")); // Predicate inferred
```

## Частые ошибки

- Перепутать arity `Func`.
- `Predicate` vs `Func<T,bool>` — interchangeable в practice.
- Public API только `Func` без domain naming.
