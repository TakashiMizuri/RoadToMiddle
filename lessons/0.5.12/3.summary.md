# Skip links, focus, contrast — краткая выжимка

> `0.5.12` · ~7 мин на повторение

## Главная идея

**Skip link** прыгает к `#main` для keyboard users; **`tabindex="-1"`** на main переносит focus. **`:focus-visible`** даёт видимый focus ring. **Contrast ratio** — измеримый минимум читаемости (AA: 4.5:1 normal text).

## Ключевые пункты

- Skip — often **first Tab stop**; visually hidden until `:focus`.
- `<main id="main" tabindex="-1">` — focus + scroll после skip.
- Не `outline: none` без **`:focus-visible`** replacement.
- **4.5:1** normal text; **3:1** large text и UI components (AA).
- Статус/error — не **только цвет** (`0.5.9`, `0.5.11`).
- DevTools / WebAIM / axe для contrast; keyboard для skip.

## Самые важные примеры

```html
<a href="#main" class="skip-link">Skip to main content</a>
<main id="main" tabindex="-1">...</main>
```

```css
:focus-visible { outline: 2px solid #005fcc; outline-offset: 2px; }
```

## Частые ошибки

- Nav-heavy page без skip link.
- Skip только scroll без focus на main.
- Global `* { outline: none }`.
- `#888` on `#777` text.

## Мнемоника

> **Skip, Stick, See** — skip to main, focus sticks on landmark, see focus ring with contrast.
