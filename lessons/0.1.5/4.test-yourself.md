## Practical (optional)

Создай repo с merge:

```bash
mkdir dag-test && cd dag-test
git init
echo base > f.txt && git add f.txt && git commit -m "root"
git checkout -b feature
echo feat >> f.txt && git add f.txt && git commit -m "on feature"
git checkout main
echo main >> f.txt && git add f.txt && git commit -m "on main"
git merge feature -m "merge feature"
git log --oneline --graph --decorate
git cat-file -p HEAD | grep parent
```
