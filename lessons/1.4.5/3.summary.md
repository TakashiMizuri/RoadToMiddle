# TaskCompletionSource — краткая выжимка

> `1.4.5` · ~10 мин

## Главная идея

**TCS** создаёт **`Task`**, который **ты** завершаешь через **`TrySetResult/Exception/Canceled`**. Мост callback/event → **`await`**.

## Ключевые пункты

- Complete **at most once** — TrySet* в handlers.
- **`RunContinuationsAsynchronously`** — safe library default.
- Many awaiters на один `tcs.Task`.
- Link **CancellationToken** для timeout/cancel.

## Пример

```csharp
var tcs = new TaskCompletionSource<int>(
    TaskCreationOptions.RunContinuationsAsynchronously);
callback => tcs.TrySetResult(data);
await tcs.Task;
```
