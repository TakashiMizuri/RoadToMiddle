# Exercises: DOCTYPE, lang, and UTF-8 charset

> Complete after reading `1.lection-eng.md`, `2.lection-ru.md` and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Minimal skeleton
**Goal:** Write a valid HTML5 boilerplate from memory.

Create `about.html` with:
- HTML5 DOCTYPE
- `lang="en"`
- UTF-8 charset as first element in `<head>`
- `<title>About Us</title>`
- One `<p>` in body

Open in browser; tab title should show correctly.

**Success criteria:** View Source shows all five required pieces; no visible encoding issues.

---

### E2. Language switch for a fragment
**Goal:** Practice nested `lang`.

Start from E1 (`lang="en"`). Add a paragraph: English intro + Russian name **Алексей** inline.

**Success criteria:** Root stays `en`; Russian fragment has its own `lang="ru"` wrapper.

---

## Medium

### M1. Cyrillic page
**Goal:** Align encoding with Russian content.

Create `team.html` entirely in Russian:
- `lang="ru"`
- Title: «Наша команда»
- Body: heading (plain text for now) and two paragraphs with Cyrillic names

Save as UTF-8. Open locally and in browser.

**Success criteria:** All Cyrillic renders correctly; charset meta present early in head.

---

### M2. Find the structural bugs
**Goal:** Analysis before validation.

Broken file provided — fix it:

```html
<html>
<head>
  <title>Broken</title>
  <h1>Welcome</h1>
  <meta charset="UTF-8">
</head>
<body>
  <p>Content</p>
</body>
</html>
```

List what is wrong and produce a corrected version.

**Success criteria:** DOCTYPE added; charset first in head; `h1` moved to body; `lang` added.

## Hard (stretch)

### H1. Encoding detective
**Goal:** Understand three-layer charset.

Write two sentences explaining what happens if:
1. File is UTF-8, meta UTF-8, HTTP header missing charset
2. File is UTF-8, meta UTF-8, HTTP declares `iso-8859-1`

Optional: serve `team.html` via a local static server and inspect response headers (browser Network tab).

**Success criteria:** Explains decoder precedence; mentions mojibake risk for case 2.

## Hints (read only if stuck)

<summary>Hint M2</summary>

</details>

<summary>Hint H1</summary>
