# Exercises: Stack Frames

## Easy
### E1. Depth limit
**Goal:** Feel stack overflow.
**Steps:** Write recursive `Count(n)` returning n at base 0; find approximate n before overflow (debug, small increments). Rewrite iterative.
**Success criteria:** Explain why iteration avoids frame growth.

## Medium
### M1. Stack diagram
**Goal:** Tie 1.1.6 to frames.
**Steps:** ASCII diagram for `void M(){ int a=1; var s="x"; var o=new object(); }` — label frame vs heap.
**Success criteria:** Three locals + one heap object.

## Hard
### H1. stackalloc bounds
**Goal:** Safe stackalloc usage.
**Steps:** Write method using `stackalloc Span<char>` buffer size 256 for parsing; explain why not size from unvalidated user input.
**Success criteria:** Mention stack limit + overflow risk.
