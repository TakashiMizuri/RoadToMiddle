## Answers

### 1. [Recall]
**Answer:** На этапе **компиляции** (merge в одно определение типа до semantic analysis).
**Why:** Partial — compile-time feature; в IL один тип.

### 2. [Recall]
**Answer:** **class**, **struct**, **record**, **interface**.
**Why:** Спецификация C#; enum не поддерживается.

### 3. [Recall]
**Answer:** Вызовы **удаляются** компилятором; метода нет в IL, runtime cost нулевой.
**Why:** Optional hook pattern для generators.

### 4. [Understanding]
**Answer:** Designer перезаписывает `.Designer.cs` при save; разделение защищает ручной код от regen.
**Why:** Tool ownership boundary.

### 5. [Understanding]
**Answer:** **Да** — части одного типа считаются одним классом для private visibility.
**Why:** Merge в единый type declaration.

### 6. [Understanding]
**Answer:** Следующая генерация **перезапишет** файл; правки потеряются.
**Why:** Generated file — артефакт build/tool, не source of truth.

### 7. [Application]
**Answer:** Имя типа, namespace, `partial`, совместимые modifiers (class/record и т.д.), один assembly.
**Why:** Иначе фрагменты не merge или duplicate type error.

### 8. [Application]
**Answer:** В **ручной** partial-файл, например `OrderDto.cs`, не в `.g.cs`.
**Why:** Generated остаётся под tool.

### 9. [Analysis]
**Answer:** Без `partial` на обоих файлах компилятор видит **два отдельных** class `Report` с дублирующим `Run` — конфликт.
**Why:** Нужен `partial` на каждом фрагменте для merge.

### 10. [Analysis]
**Answer:** Правило **исчезнет** при regen designer; возможны merge conflicts или overwrite.
**Why:** Designer owns that file.

### 11. [Understanding]
**Answer:** Partial — **один** тип в двух файлах; наследование — **два** типа в иерархии, base может быть не partial-friendly.
**Why:** Partial — idiomatic для same-type extension без hierarchy.

### 12. [Application]
**Answer:** В своём partial `DbContext` — `OnModelCreating` или `IEntityTypeConfiguration` в отдельных файлах.
**Why:** Scaffold трогает только generated partial.

### 13. [Recall]
**Answer:** **Нет.**
**Why:** Partial не применяется к enum.

### 14. [Understanding]
**Answer:** Partial method без impl **вырезается** — не нужен empty override; generated не делает virtual hierarchy.
**Why:** Zero-cost optional extension.

### 15. [Application]
**Answer:** **`1.3.1` — Generics** (узел 1.3).
**Why:** Up Next после 1.2.18 по curriculum.
