# SHA-1, commit graph, parent pointers — краткая выжимка

> `0.1.5` · ~8 мин на повторение

## Главная идея

Каждый Git object идентифицируется **hash** (SHA-1 по умолчанию) от content. **Commit'ы** связаны **parent pointers** в **DAG** (Directed Acyclic Graph). **HEAD** определяет **reachable** history. Hash'и обеспечивают **integrity** (tamper-evident chain) и **deduplication** (один content — один object).

## Ключевые пункты

- **Commit fields:** `tree`, `parent`(s), `author`, `committer`, `message`.
- **Root commit** — без parent; **merge commit** — 2+ parent'ов.
- **First parent** при merge — ветка, на которой выполняли `git merge`.
- **DAG:** directed (child → parent), acyclic (нет циклов), graph (ветки и merge).
- **Reachability:** от HEAD/refs по parent'ам назад — что видит `git log`.
- **SHA-256:** опционально с Git 2.29+ (`git init --object-format=sha256`).
- Rebase/cherry-pick → новый commit hash (меняются parent/tree/metadata).

## Самые важные примеры

**Merge commit (два parent'а):**
```
parent abc...   ← main до merge
parent def...   ← tip влитой ветки
```

**Визуализация:**
```bash
git log --oneline --graph --decorate --all
```

**First / second parent:**
```bash
git rev-parse HEAD^     # first parent
git rev-parse HEAD^2    # second parent (только merge)
```

## «Под капотом» в одном абзаце

Git сериализует commit body, добавляет header `commit <size>\0`, считает SHA-1 — это hash object. Поле `parent` внутри body включается в hash, поэтому commit **криптографически привязан** к предкам. Обход графа (log, merge-base, rebase) — это traversal DAG по parent links. Unreachable commit'ы остаются на диске до gc, если на них указывает reflog или другой ref.

## Частые ошибки

- Думать, что hash «случайный» — он deterministic.
- Путать merge (два **parent link**) с двумя копиями файлов.
- Считать, что удаление branch сразу удаляет commit'ы.
- Не различать author и committer после rebase.

## Мнемоника

> **Hash = fingerprint content.** **Parent = стрелка назад во времени.** **HEAD = откуда смотрим на граф.**
