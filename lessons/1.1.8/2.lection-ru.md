## Core Concepts

### Арифметические операторы

| Operator | Meaning |
|----------|---------|
| `+` `-` `*` `/` `%` | Стандартная арифметика |
| `++` `--` | Increment/decrement |

**Integer division:**

```csharp
int a = 5 / 2;           // 2
double b = 5 / 2;        // 2.0 — сначала int division!
double c = 5.0 / 2;      // 2.5
```

Деньги — **`decimal`** и суффикс **`m`** (`1.1.3`):

```csharp
decimal tax = total * 0.075m;
```

**Overflow:** в **`unchecked`** (default) integer wraparound; в **`checked`** — **`OverflowException`**.

### Операторы сравнения

`==`, `!=`, `<`, `>`, `<=`, `>=`.

Value types — по значению (если нет overload). Reference types — **identity** по default, кроме overload (`string` сравнивает content).

```csharp
ReferenceEquals(s1, s2);  // разные objects
s1 == s2;                 // true для одинакового content string
```

**Не используй `==` для `double`/`float`** без epsilon; для денег — `decimal`.

**NaN:** `NaN == NaN` → false; `double.IsNaN(x)`.

### Логические операторы

| `&&` | AND | short-circuit |
| `\|\|` | OR | short-circuit |
| `!` | NOT | |

```csharp
if (user != null && user.IsActive) { }
```

**`&` `|` на bool** — без short-circuit; для flags — bitwise (`1.1.17`).

### Null-coalescing

**`??`:** если слева не null — результат слева, иначе справа:

```csharp
string name = input ?? "Anonymous";
string s = a ?? b ?? "default";   // left-associative
```

**`??=`:** присвоить только если null:

```csharp
list ??= new List<int>();
```

Работает с **`int?`**, **`string?`**, reference types.

```csharp
// int x = 0 ?? 1;  // ошибка
int? x = 0;
int y = x ?? 1;       // 0
```

### Ternary и precedence

```csharp
string status = score >= 70 ? "Pass" : "Fail";
```

При сомнении — **скобки**.

## Under the Hood

Mixed numeric types — **numeric promotion** по spec C#. IL: `add`, `div`, `ceq`; **`checked`** → `add.ovf`.

`&&`/`||` → conditional branches; RHS может не выполниться.

`??` → null check / `HasValue` для `Nullable<T>` без boxing для `int? ?? 0`.

## Common Mistakes & Anti-patterns

Int division вместо real. `==` для double. `&` вместо `&&` с null. Перегруженный nested ternary.

---

## Production & Real-World Notes

Configuration defaults, validation chains, порядок cheap checks в `&&`. NRT + `??` снижают NRE.

---

## Comparison / Trade-offs

`??` — default при null; ternary — две ветки по условию; `if` — side effects (`1.1.9`); `decimal` — деньги; `checked` — критичная арифметика.

## Quick Reference

```
5/2 → 2
??  default if null
&& || short-circuit
checked → throw on overflow
```

## Key Takeaways

- **Int `/` truncates.**
- **`m` для decimal literals.**
- **Reference `==`** — identity unless overloaded.
- **Short-circuit** — порядок operands важен.
- **`??` / `??=`** — null-safe defaults.
