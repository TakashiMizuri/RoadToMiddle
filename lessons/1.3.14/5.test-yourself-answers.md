## Answers

### 1. [Recall]
**Answer:** (1) static class, (2) static method, (3) первый параметр с модификатором `this`.
**Why:** Спецификация C# для extensions.

### 2. [Recall]
**Answer:** `StringEx.OrEmpty("hi")` (или фактическое имя класса).
**Why:** Compile-time lowering; receiver — первый аргумент.

### 3. [Recall]
**Answer:** `System.Runtime.CompilerServices.ExtensionAttribute`.
**Why:** IDE/analyzers используют для подсказок instance-style.

### 4. [Understanding]
**Answer:** Extension — static call; метод **входит** с `this == null` и может обработать. Instance `Length` разыменовывает null receiver **до** вызова property getter → NRE.
**Why:** Нет implicit null-check на receiver для extensions.

### 5. [Understanding]
**Answer:** `IEnumerable<T>` — BCL interface; нельзя добавить instance members post-hoc без breaking change. Extensions в отдельной static class (`System.Linq`) — additive, optional `using`.
**Why:** Open-closed для BCL; один `Enumerable` для всех implementers.

### 6. [Understanding]
**Answer:** **Instance method** на типе `Customer`.
**Why:** Instance always beats extension при совпадении сигнатуры.

### 7. [Application]
**Answer:** `using System.Linq;`
**Why:** `Where` — extension в `Enumerable`.

### 8. [Application]
**Answer:** **Нет** (из другой assembly), если поле private и нет `InternalsVisibleTo`.
**Why:** Extension — обычный static method без special access.

### 9. [Application]
**Answer:** Static call: `ClassA.Format(42)` или `ClassB.Format(42)`.
**Why:** Explicit static invocation bypasses extension sugar.

### 10. [Analysis]
**Answer:** Extensions **не** меняют vtable; dispatch static, metadata только ExtensionAttribute. Vtable — для virtual instance methods.
**Common mistake:** Путать syntactic sugar с runtime monkey-patching.

### 11. [Analysis]
**Answer:** **Не компилируется** — нет доступа к private `_repository` из static extension (если не nested/other trick).
**Why:** Extensions не получают elevated access.

### 12. [Understanding]
**Answer:** Desugar в цепочку extension calls: `xs.Where(...).Select(...)` через `Enumerable`.
**Why:** Query syntax — compiler transform.

### 13. [Application]
**Answer:**
```csharp
public static string ToCurrency(this decimal amount, string culture);
```
(in static class, static method)
**Why:** Standard extension shape.

### 14. [Recall]
**Answer:** **Да**, constraints на type parameter extension допустимы.
**Why:** Generic static method rules.

### 15. [Understanding]
**Answer:** Runtime видит **реальный** static method; instance syntax только compile-time.
**Why:** No vtable entry; debugging shows static name.
