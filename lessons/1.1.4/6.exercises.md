# Exercises: string

> Complete after lectures and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Literal gallery
**Goal:** Master `@`, `$`, escapes.

Create constants (no runtime input):
1. Regular string with escaped quote: `She said "Hi"`
2. Verbatim Windows path: `C:\Temp\logs`
3. Interpolated: name + year → `"Roadmap 2026"`
4. Combined: `$@"C:\Users\{user}\Desktop"` with `user = "dev"`

Print all four.

**Success criteria:** Compiles; paths display correct backslashes.

---

### E2. Immutability proof
**Goal:** Observe reference vs content.

```csharp
string s1 = "test";
string s2 = s1;
s1 = s1.ToUpperInvariant();
```

Print `s1`, `s2`. Explain in one sentence why `s2` unchanged.

**Success criteria:** Output `TEST` and `test`; explanation mentions immutability + new object.

## Medium

### M1. Join and split
**Goal:** API fluency.

Given `string csv = "alpha;beta;gamma;"`, split on `;`, remove empty entries, rejoin with ` | `. Print result.

**Success criteria:** `alpha | beta | gamma`

---

### M2. Intern experiment
**Goal:** See ReferenceEquals behavior.

```csharp
string litA = "intern-me";
string litB = "intern-me";
string built = string.Concat("intern", "-", "me");
Console.WriteLine(ReferenceEquals(litA, litB));
Console.WriteLine(ReferenceEquals(litA, built));
Console.WriteLine(ReferenceEquals(litA, string.Intern(built)));
```

Run and record three lines. Explain each result in 2–3 sentences.

**Success criteria:** Three bool outputs explained with interning rules.

## Hard (stretch)

### H1. Concat vs builder preview
**Goal:** Feel allocation pattern (preview `1.1.16`).

Build string of 5 000 `x` characters two ways:
- A: loop with `s += "x"`
- B: `StringBuilder` loop then `ToString()`

Measure elapsed time with `Stopwatch` (rough is fine). Which is faster? Why in theory?

**Success criteria:** B significantly faster; paragraph cites repeated copy on `+=`.

---

### H2. Culture formatting
**Goal:** Invariant vs current.

```csharp
decimal n = 1234.56m;
Console.WriteLine(n.ToString(CultureInfo.CurrentCulture));
Console.WriteLine(n.ToString(CultureInfo.InvariantCulture));
```

Run under your OS locale. Document difference. When would you use each in ASP.NET Core API?

**Success criteria:** Two different formats shown; API → Invariant or explicit format noted.
