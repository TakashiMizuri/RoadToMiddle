# Exercises: Namespaces and Using

> Complete after reading lectures and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. File-scoped layout
**Goal:** Split types by namespace.

In a new console project `NsLab`, create:

- `Core/Messages.cs` — namespace `NsLab.Core`, static class `Messages` with `public const string Welcome = "Hi";`
- `Program.cs` — use `using NsLab.Core;` and print `Messages.Welcome`

**Success criteria:** Build and run print `Hi`.

---

### E2. Alias drill
**Goal:** Practice `using alias`.

Add a file `Demo/AliasDemo.cs` with namespace `NsLab.Demo`. Without adding NuGet packages, write **two** alias lines that would disambiguate `JsonSerializer` from `System.Text.Json` vs a hypothetical `Other.JsonSerializer` (second can be a stub nested class in same file). Call both in a method `Compare()`.

**Success criteria:** Compiles; comments explain which alias maps to which type.

---

## Medium

### M1. Implicit usings off
**Goal:** See what ImplicitUsings provides.

Set in `.csproj`:
```xml
<ImplicitUsings>disable</ImplicitUsings>
```

Fix `Program.cs` with explicit `using` until it builds. List every `using` you added.

**Success criteria:** Build succeeds; list of usings documented.

---

### M2. Folder vs namespace
**Goal:** Convention vs compiler.

Create `Features/Billing/Invoice.cs` with namespace `NsLab.Features.Billing` and record `Invoice(decimal Amount)`.

Create **intentional mismatch**: `Features/Billing/Legacy.cs` with namespace `NsLab.Legacy` (file still under Billing folder). Reference both from `Program.cs`.

**Success criteria:** Builds; one paragraph explaining that folder path ≠ namespace requirement.

---

## Hard (stretch)

### H1. GlobalUsings.cs
**Goal:** Project-wide imports.

Re-enable ImplicitUsings **or** add manual `GlobalUsings.cs`:

```csharp
global using System;
global using NsLab.Core;
```

Remove duplicate file-level usings from `Program.cs` where redundant. Verify build.

**Success criteria:** `Program.cs` shorter; single global import file; project compiles.
