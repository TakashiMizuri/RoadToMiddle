# LINQ Aggregate, Scan — краткая выжимка

> `1.3.23` · ~8 мин

## Главная идея

**`Aggregate`** сворачивает sequence в **одно** значение (fold). **`Scan`** (custom) yield'ит **каждое промежуточное** состояние accumulator.

## Ключевые пункты

- No-seed: first element = acc; **empty throws**.
- Seed: **empty → seed**; тип acc может ≠ element.
- **`resultSelector`**: project final acc без extra variable.
- Prefer **`Sum`/`Max`/`ToDictionary`** когда есть.
- **`Scan`**: not in System.Linq; MoreLINQ or yield implementation.
- **`Aggregate`** always full enumeration; immediate.

## Примеры

```csharp
nums.Aggregate(1, (a,n) => a * n);           // product, empty→1
nums.Aggregate(0, (a,n) => a + n);           // sum
Scan(nums, 0, (a,n) => a + n);                 // 1,3,6 running
```

## «Под капотом»

Plain foreach; EF rarely translates. String + in fold allocates quadratically.
