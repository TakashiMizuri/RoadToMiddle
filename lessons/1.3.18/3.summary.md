# Index и Range — краткая выжимка

> `1.3.18` · ~8 мин на повторение

## Главная идея

**`^n`** — index from end (`^1` = last). **`start..end`** — slice с **exclusive end**. Работает на array, string, Span — замена `Length - 1` arithmetic.

## Ключевые пункты

- `arr[0]`, `arr[^1]`, `arr[^2]` — с начала / с конца.
- **`^0` invalid** — exception.
- `[1..3]` — indices 1,2 only; `[2..]`, `[..2]`, `[..]` — open bounds.
- **End exclusive** — `[0..3]` length 3.
- **String** slice → new string (allocate); **Span** → view.
- `Range`/`Index` as variables: `1..^1` excludes first and last.
- LINQ Skip/Take — lazy; array range — copy segment.

## Самые важные примеры

```csharp
int[] a = { 1, 2, 3, 4 };
a[^1];      // 4
a[1..3];    // { 2, 3 }
a[1..^1];   // { 2, 3 } — без first и last

string s = "hello";
s[^1..];    // "o"
s[..2];     // "he"
```

## Частые ошибки

- Думать, что end inclusive.
- `^0` или `^n` на короткой sequence.
- Range на IEnumerable без ToArray.

## Мнемоника

> **`..` как [start, end) в математике — end не входит.**
