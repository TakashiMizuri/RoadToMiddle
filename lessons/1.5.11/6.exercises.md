# Exercises: Memory Leaks

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Event leak repro
**Goal:** Console app: static event, create 1000 subscribers, no `-=`; observe Gen2 growth (`GC.GetTotalMemory`).
**Success criteria:** Growth plateaus high; fix with `-=` on destroy — memory drops after GC.

### E2. Stored lambda unsubscribe
**Goal:** Subscribe with named `EventHandler` field; unsubscribe in `Dispose`.
**Success criteria:** Subscriber collectable after Dispose + GC.

## Medium

### M1. Singleton scope factory
**Goal:** Singleton `Worker` uses `IServiceScopeFactory` to process item with scoped `FakeDb`.
**Success criteria:** No scoped field on singleton.

### M2. Bounded cache
**Goal:** Replace unbounded `static Dictionary` with `MemoryCache` size limit 100.
**Success criteria:** 1000 inserts — count ≤ 100.

## Hard (stretch)

### H1. dotnet-gcdump diff
**Goal:** Collect dump before/after leak repro; identify retaining type path (report summary).
**Success criteria:** One paragraph retention chain in README.

## Submission checklist

- [ ] One `-=` fix demonstrated
- [ ] One static root identified
- [ ] Note on singleton vs scoped
