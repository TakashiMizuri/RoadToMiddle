# Exercises: Merge vs Rebase

## Easy

### E1. Policy picker
Pick merge / rebase / squash for: (a) integrate to `main`, (b) update feature with `main`, (c) shared `release/2.0`. One sentence each.

## Medium

### M1. Write CONTRIBUTING snippet
Draft 6–10 lines: merge method on `main`, rebase rules, force-push rules.

## Hard

### H1. Debate moderation
Two devs argue rebase-only vs merge-only. Write neutral  paragraph with decision criteria (team size, audit, release cadence).

<details><summary>Hint E1 (c)</summary>Shared release — merge/cherry-pick, not rebase tip.</details>
