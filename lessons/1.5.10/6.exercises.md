# Exercises: Dispose Pattern

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Sealed TempFile
**Goal:** `TempFile : IDisposable` creates temp file, deletes on Dispose; idempotent.
**Success criteria:** Double Dispose safe; no finalizer.

### E2. SuppressFinalize demo
**Goal:** Type with finalizer + `Dispose` calling `SuppressFinalize`; log which path ran.
**Success criteria:** After `using`, finalizer does **not** run (use `GC.KeepAlive` if testing — explain).

## Medium

### M1. Full pattern base + derived
**Goal:** `NativeHolderBase` with fake `IntPtr`, `DerivedHolder` with `MemoryStream` field.
**Success criteria:** Correct `Dispose(bool)` split; derived calls base.

### M2. SafeHandle wrapper
**Goal:** Minimal `SafeHandle` subclass; outer sealed type disposes handle only via `Dispose`.
**Success criteria:** Outer class has **no** finalizer.

### M3. ObjectDisposedException
**Goal:** Public method on disposable type throws after Dispose (.NET 7+ `ThrowIf`).
**Success criteria:** Clear API contract in XML comment.

## Hard (stretch)

### H1. Leak vs Dispose benchmark
**Goal:** Open many handles without Dispose until failure; repeat with `using`.
**Success criteria:** Tie to **`1.3.16`** consumer practices in README note.
