# Exercises: Extension Methods

> Complete after passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. First extension
**Goal:** `IsPositive(this int n)` returning bool.
**Steps:** Static class `IntExtensions`, test `(-1).IsPositive()` and `5.IsPositive()`.
**Success criteria:** Instance-style call; compiles to static call (verify in IDE Go To Definition → static class).

### E2. Null-tolerant string
**Goal:** `OrDefault(this string? s, string fallback)`.
**Steps:** Call on `null` without `?.`.
**Success criteria:** No NRE; documents null behavior in XML comment.

### E3. Missing using
**Goal:** Reproduce compile error without `using System.Linq`, then fix.
**Success criteria:** Explain why `List<T>` alone is insufficient.

## Medium

### M1. Fluent builder
**Goal:** Extensions on `StringBuilder`: `AppendLineIf(this StringBuilder sb, bool cond, string line)`.
**Steps:** Chain three calls in one expression returning same builder.
**Success criteria:** Returns `StringBuilder` for chaining.

### M2. Instance vs extension
**Goal:** Class `Widget { public string Format() => "instance"; }` + extension `Format(this Widget w) => "extension"`.
**Steps:** Predict and verify output of `new Widget().Format()`.
**Success criteria:** Explain instance precedence.

### M3. Generic clamp
**Goal:** `Clamp<T>(this T value, T min, T max) where T : IComparable<T>`.
**Success criteria:** Works for `int` and `decimal` with same extension.

## Hard (stretch)

### H1. Decompile or ILSpy
**Goal:** Build console app calling your extension; inspect IL for `ExtensionAttribute` and static `call`.
**Success criteria:** Screenshot or describe IL opcodes — no instance callvirt on extended type.

### H2. Ambiguity fix
**Goal:** Two extensions `Show(this int)` in different namespaces; resolve via explicit static call only in one site.
**Success criteria:** Project compiles; no CS0121 at ambiguous site.

## Submission checklist

- [ ] At least one null-tolerant extension
- [ ] Demonstrated `using System.Linq` requirement
- [ ] Can explain lowering in one sentence
