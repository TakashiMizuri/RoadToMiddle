# Exercises: LINQ Take, Skip, TakeWhile, SkipWhile

> Complete after lectures and passing `4.test-yourself.md`.

## Easy

### E1. Page slice
**Goal:** Implement `GetPage(IEnumerable<int> source, int page, int size)` returning 1-based page items.
**Success criteria:** Page 2 size 3 on `1..10` → `4,5,6`.

### E2. TakeWhile prefix
**Goal:** Return longest initial run of positive numbers from `IEnumerable<int>`.
**Success criteria:** `[3,5,-1,2]` → `[3,5]`; uses early stop.

## Medium

### M1. Skip CSV comments
**Goal:** Parse lines, skip `#` comments and blank lines at start, then take until `---END---`.
**Success criteria:** `SkipWhile` + `TakeWhile` composed correctly.

### M2. EF pagination DTO
**Goal:** Sketch `GetProductsPageAsync(page, size)` with `OrderBy`, `Skip`, `Take`, `CountAsync`.
**Success criteria:** Stable sort; total count without double materialization pitfall explained.

## Hard (stretch)

### H1. Keyset vs offset
**Goal:** Rewrite offset pagination as `WHERE Id > @lastId ORDER BY Id TAKE size` pseudo-LINQ.
**Success criteria:** Paragraph when keyset wins over `Skip`.
