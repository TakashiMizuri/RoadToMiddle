# Exercises: Collections intro

> Complete after lectures and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Shopping list
**Goal:** Basic List operations.
**Steps:** `List<string> items`; Add 3 items; Remove one; print Count and each item.
**Success criteria:** Uses Add, Remove/RemoveAt, foreach.

### E2. Phone book
**Goal:** Dictionary add and read.
**Steps:** `Dictionary<string, string>` name → phone; add 3; `TryGetValue` for existing and missing.
**Success criteria:** No KeyNotFoundException on miss path.

## Medium

### M1. Unique visitors
**Goal:** HashSet dedup.
**Steps:** List of visitor ids with duplicates; count unique with HashSet; print Add return values for duplicates.
**Success criteria:** Correct unique count.

### M2. Word frequency
**Goal:** Dictionary as counter.
**Steps:** Split sentence into words; foreach word increment `dict[word]` or TryGetValue pattern.
**Success criteria:** O(n) single pass; correct counts.

### M3. List to index
**Goal:** Build lookup from list.
**Steps:** `List<(int Id, string Name)>` → `Dictionary<int, string>`; lookup by id in loop.
**Success criteria:** O(1) average lookups after build.

## Hard (stretch)

### H1. Group by category
**Goal:** `Dictionary<string, List<string>>`.
**Steps:** Items with category; group names per category; print one category's list.
**Success criteria:** Lists created on first sight of category; no null lists.

### H2. Safe list cleanup
**Goal:** Remove negatives without foreach trap.
**Steps:** `list.RemoveAll(x => x < 0)` or reverse for loop; explain why foreach+Remove fails.
**Success criteria:** Correct result; written explanation.

## Submission checklist

- [ ] E1–E2 run
- [ ] M2 frequency correct
- [ ] H1 grouping works
