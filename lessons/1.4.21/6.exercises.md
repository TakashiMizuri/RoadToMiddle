# Exercises: TaskScheduler and SynchronizationContext

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Predict resume thread
**Goal:** Mental model.

**Steps:** For each: (a) Console app `await Delay`, (b) WPF button `await Delay`, (c) ASP.NET Core action `await Delay` — where does continuation run?

### E2. ConfigureAwait experiment
**Goal:** Library vs app.

**Steps:** Pseudocode library method with and without `ConfigureAwait(false)`. Explain which caller scenarios differ.

## Medium

### M1. Fix WPF deadlock
**Goal:** Apply pattern.

**Steps:** Rewrite sync `void Click` using `.Result` to async event handler with proper exception handling.

### M2. Shared library rule
**Goal:** Document team standard.

**Steps:** Write 5-bullet guideline for `ConfigureAwait` in internal NuGet used by API + WPF client.
