### 2. [Recall]

### 3. [Recall]
**Answer:** **`0.5m`** (decimal) или `0.5d` / `0.5f` для других типов; для денег — **`m`**.

### 4. [Understanding]
**Answer:** **`5 / 2` выполняется как int division** (оба int) → `2`; затем implicit conversion to `double` → `2.0`. Дробная часть теряется **до** conversion.

### 5. [Understanding]
**Answer:** Если `a == null`, **`a.Length > 0` не evaluates** — short-circuit `&&` stops at false LHS.

### 6. [Understanding]
**Answer:** **`|`** evaluates **both** sides always; **`||`** short-circuits — skips RHS if LHS true.

### 7. [Application]
**Answer:** Tricky postfix/prefix. Start `x=10`. `x++` uses 10 then x=11; `++x` makes x=12, uses 12; sum **y=22**, final **x=12**. (Verify in small program if unsure.)

### 8. [Application]
**Answer:** **`decimal discount = coupon ?? 0m;`** (assuming `coupon` is `decimal?`).

### 9. [Application]
**Answer:** Заменить **`||` на `&&`**: `if (items != null && items.Count > 0)`. С `||` при `items == null` всё равно evaluate `items.Count` → NRE.

### 10. [Application]
**Answer:** **`7 % 3 == 1`**. **`(-7) % 3 == -1`** (remainder sign follows dividend in C#).
