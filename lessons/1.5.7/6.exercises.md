# Exercises: Generations

## Easy
### E1. Generation check
**Goal:** Observe Gen after alloc.
**Steps:** Console: `var o = new object(); Console.WriteLine(GC.GetGeneration(o));` — result?
**Success criteria:** Gen0 for small SOH object.

## Medium
### M1. Server GC verify
**Goal:** Know your host config.
**Steps:** Check `.csproj` or `runtimeconfig` for ServerGarbageCollection; explain default for ASP.NET template.
**Success criteria:** true for web template.

## Medium
### M2. Allocation audit plan
**Goal:** Connect Gen0 rate to code.
**Steps:** Outline 5 steps to investigate high gen-0-gc-count in API (tools + code areas).
**Success criteria:** Include dotnet-counters, allocation profiling, JSON/EF.
