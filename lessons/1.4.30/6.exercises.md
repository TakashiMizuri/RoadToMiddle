# Exercises: Fire-and-Forget Task

> Complete after passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Smell Detector
**Goal:** List 5 code patterns in a codebase that indicate unsafe fire-and-forget (synthetic snippets OK).
**Success criteria:** Each with one-line risk (scope, exception, pool).

### E2. Scope Sketch
**Goal:** Pseudocode `ProcessQueueItem` that creates DI scope, resolves `IMailer`, awaits send, disposes scope.
**Success criteria:** `try/finally` or `await using` on scope.

## Medium

### M1. Refactor Controller
**Goal:** Rewrite a controller that uses `_ = NotifyAsync()` to enqueue work; keep `Accepted()` response.
**Success criteria:** No captured request-scoped fields in closure.

### M2. Observation Bridge
**Goal:** Implement extension `FireAndForgetWithLog(this Task task, ILogger log)` — document what it does NOT fix.
**Success criteria:** `ContinueWith` only on `Faulted`; comment on scope limitation.

## Hard (stretch)

### H1. Outbox Mini-Design
**Goal:** Tables `Orders` + `OutboxMessages`; worker polls; idempotent handler. Diagram ASCII.
**Success criteria:** Same transaction for order + outbox row; at-least-once delivery note.
