# Exercises: Records

> Complete after lectures and passing `4.test-yourself.md`.

## Easy

### E1. Positional DTO
**Goal:** Create `public record ProductDto(int Id, string Name, decimal Price)` and demonstrate `==` between two instances with same values.
**Success criteria:** Equality true; `ReferenceEquals` false.

### E2. with update
**Goal:** Given `UserDto(int Id, string Name, string Role)`, promote user with `with { Role = "Admin" }` without mutating original.
**Success criteria:** Original `Role` unchanged; new instance has `Admin`.

## Medium

### M1. Immutable cart
**Goal:** `record Cart(IReadOnlyList<CartLine> Lines)` with `AddLine` returning new `Cart` via `with` and new list.
**Success criteria:** No in-place list mutation; prior cart unchanged.

### M2. JSON round-trip
**Goal:** Deserialize JSON to positional record command; serialize back.
**Success criteria:** `System.Text.Json` round-trip preserves equality.

### M3. record struct pixel
**Goal:** `readonly record struct Color(byte R, byte G, byte B)` with method `ToHex()` returning `#RRGGBB`.
**Success criteria:** Struct copy by value; no heap allocation in simple loop (conceptual).

## Hard (stretch)

### H1. Fix shallow copy bug
**Goal:** Start with `record Config(Dictionary<string,string> Settings)` showing shared mutation bug; fix with `ImmutableDictionary` or defensive copy in constructor.
**Success criteria:** `with` clone cannot mutate sibling's settings.

### H2. Record inheritance
**Goal:** `abstract record Shape(string Name)` + `sealed record Circle(double Radius) : Shape("circle")` — demonstrate equality includes base property.
**Success criteria:** Two circles with same radius equal; different radius not equal.
