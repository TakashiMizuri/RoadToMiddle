# Custom IEnumerable с yield — краткая выжимка

> `1.3.29` · ~8 мин

## Главная идея

**`yield return`** создаёт **lazy cold IEnumerable**: producer отдаёт элементы по pull consumer'а; compiler генерирует **state machine**.

## Ключевые пункты

- `foreach` → `GetEnumerator` + `MoveNext` + `Dispose`.
- **`yield return`** — pause/resume; **`yield break`** — end.
- Custom iterators compose с LINQ без `List`.
- Each enumeration cold source — **re-run** logic (I/O twice = 2× cost).
- `try/finally`/`using` в iterator — cleanup при early stop.
- **`IAsyncEnumerable`** — async streaming variant.

## Примеры

```csharp
while (true) yield return n++;
ReadLines(path).Where(...).Take(100);
Paginated API: yield pages until Take satisfied.
```

## «Под капотом»

Roslyn → class с `_state`, locals → fields; each `MoveNext` switch по state. Independent enumerators = independent machines.

## Частые ошибки

- Double enumeration без cache.
- Infinite iterator без bound.
- Mutable shared state.
