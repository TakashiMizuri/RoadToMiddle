### 1. MoveNext(); SetStateMachine(IAsyncStateMachine).
### 2. Continuation must hold stable reference; struct on stack would be invalid after return — box/copy to heap.
### 3. Registers state machine continuation on awaiter; MoveNext called when awaited task completes.
### 4. Terminal — method finished (before/after SetResult marker in generator).
### 5. OK — suspended, waiting completion; SetResult on resume when done.
### 6. catch block in MoveNext → builder.SetException(ex).
### 7. default (-1) + case 0 — two segments; second entry after completion.
### 8. Stack frame torn down at return from MoveNext; resume needs persisted awaiters/locals.
### 9. Not bug — compiler-generated; PDB maps to source await lines.
### 10. SharpLab or dotnet build /p:EmitCompilerGeneratedFiles=true → obj/generated.
