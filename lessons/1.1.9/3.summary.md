# Control flow: if/else, switch — краткая выжимка

> `1.1.9` · ~8 мин на повторение

## Главная идея

**Control flow** направляет выполнение по **boolean conditions**. После операторов (`1.1.8`) ты используешь `if/else`, ternary и `switch` (classic + expression), чтобы программа реагировала на данные, а не шла линейно.

## Ключевые пункты

- `if` / `else if` / `else` — цепочка проверок; выполняется **первая** истинная ветка.
- Condition всегда **`bool`**; C# без «truthy» чисел.
- **`&&` / `||`** — short-circuit; слева guard для справа (`name != null && name.Length`).
- **Ternary** `? :` — выбор **value**, не сложной логики.
- **Classic `switch`**: `case`, **`break`**, `default`; без fall-through как в C.
- **Switch expression**: `x switch { pattern => result, _ => ... }` — возвращает value.
- **Patterns**: constant, relational (`>= 90`), type (`string s`), `or`, discard `_`.

## Самые важные примеры

```csharp
string label = score >= 60 ? "Pass" : "Fail";

string tier = score switch
{
    >= 90 => "Gold",
    >= 70 => "Silver",
    _     => "Bronze"
};
```

```csharp
if (!int.TryParse(text, out int age)) return;
if (age < 18) Console.WriteLine("Minor");
else Console.WriteLine("Adult");
```

## Частые ошибки

- Неверный **порядок** `else if` (90 попадает в «B»).
- **Assignment** `=` вместо `==` в condition.
- Switch по **float** для exact match.
- Глубоко вложенный **ternary** вместо `if`/`switch`.

## Мнемоника

> **If — правило; switch — таблица решений; expression — таблица с результатом.**
