## Answers

### 1. [Recall]
**Answer:** **Authoritative source** контента — S3, blob storage, web server, откуда CDN fetch'ит при miss.

---

### 2. [Recall]
**Answer:** **Hit** — object есть на edge и fresh → ответ с PoP. **Miss** — нет/stale → fetch origin, store, respond.

---

### 3. [Understanding]
**Answer:** Hash в имени = **новый URL при каждом deploy** → long cache safe. `index.html` **без hash** — тот же URL; long cache = users не получат новые script references.

---

### 4. [Understanding]
**Answer:** Разные **host** (и возможно scheme/port) → **different origins** по same-origin policy. Browser enforce CORS.

---

### 5. [Application]
**Answer:** (1) **Stale JS** still cached at edge/browser (old hash URL cached). (2) **`index.html` cached** at edge with old script tags. (3) Browser cache без revalidation.

---

### 6. [Understanding]
**Answer:** **CDN** cache'ит **static/cacheable content** близко к user. **LB** распределяет **dynamic requests** к compute instances. Разные цели.

---

### 7. [Recall]
**Answer:** Cache **1 year**; **immutable** — не revalidate while fresh (для unchanging URL).

---

### 8. [Application]
**Answer:** `/assets/*.js`, `*.css`, images, fonts с hashed names — **not** `index.html` (short TTL).

---

### 9. [Analysis]
**Answer:** **Personalized/user data** может отдаваться **wrong user** из shared edge cache; **security/privacy** breach; violates cache assumptions.

---

### 10. [Understanding]
**Answer:** **CNAME** `app.example.com` → CDN domain; client resolves to nearest PoP IP.

---

### 11. [Application]
**Answer:** Response served from **CloudFront edge cache**, not origin fetch for this request.

---

### 12. [Analysis]
**Answer:** Low traffic, single region — **complexity/cost/DNS** не окупаются; nginx local enough; cache bugs overhead.

---

### 13. [Understanding]
**Answer:** **Forced removal/refresh** cached objects at edge. Нужна когда URLs не меняются (`index.html`, non-hashed assets) после deploy.

---

### 14. [Recall]
**Answer:** **`Cache-Control`**, **`Age`**, **`X-Cache`**, **`CF-Cache-Status`**, **`ETag`**.

---

### 15. [Application]
**Answer:** `https://app.example.com` — CDN static SPA; `https://api.example.com` — dynamic API (ALB → Kestrel).
