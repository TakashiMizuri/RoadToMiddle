# CancellationToken — краткая выжимка

> `1.4.11` · ~10 мин

## Главная идея

**Cooperative cancellation** через **`CancellationToken`** — код сам проверяет flag или получает throw. **`CTS`**, **`Register`**, **`CreateLinkedTokenSource`**, **`CancelAfter`** — основной toolkit.

## Ключевые пункты

- `None` / `default` — never cancels.
- `ThrowIfCancellationRequested()` → `OperationCanceledException`.
- Linked — cancel if **any** source fires.
- `Register` — cleanup; короткие callbacks.
- Dispose CTS — особенно с `CancelAfter`.
- ASP.NET: `CancellationToken` = `RequestAborted`.
- State machine: cancel → exception path `MoveNext` (`1.4.7`).
