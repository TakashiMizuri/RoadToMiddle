# Constructors — краткая выжимка

> `1.2.3` · ~8 мин на повторение

## Главная идея

**Constructor** — special method при `new`: переводит instance из allocated memory в **valid state**. Имя = class name, без return type.

## Ключевые пункты

- Нет явных constructors → compiler даёт **parameterless default**; после первого explicit — default только если объявишь сам.
- **`: this(args)`** — chain на sibling ctor **same class**; validation в одном месте.
- **`: base(args)`** — chain на base class ctor (`1.2.6`).
- **Object initializer** `{ Prop = x }` — после ctor, в init context для `init` props.
- **Primary constructor** (C# 12): `class Point(int x, int y)` — synthesized ctor, DI-friendly.
- **Private ctor + static factory** — контроль creation и named intent.

## Самые важные примеры

```csharp
public Rectangle(double side) : this(side, side) { }
public OrderService(IOrderRepository repo) { ... }  // primary
public static Money FromMajorUnits(decimal d) => new(d);
```

## Частые ошибки

- God constructor с 20 параметрами.
- Circular `: this()` chain.
- Mutable injected dependency через primary ctor capture.
