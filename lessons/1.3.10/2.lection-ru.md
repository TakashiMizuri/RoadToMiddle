# Immediate execution — ToList, Count, First

> Roadmap: `1.3.10` · Node: `1.3` — C#: продвинутые темы · Depth: понимание

## Learning Objectives

После этой лекции ты сможешь:

- Классифицировать LINQ operators: **deferred** vs **immediate** (terminal).
- Materialize через **`ToList`**, **`ToArray`**, **`ToDictionary`**.
- Использовать **`Count`**, **`Any`**, **`All`** осознанно.
- Выбирать **`First`** vs **`FirstOrDefault`** по intent (`1.3.27` позже).
- Объяснить **`Count()`** O(1) на **`ICollection`** vs full scan через **`Where`**.
- Materialize на **правильной границе** в ASP.NET Core.
- Связать с EF **`ToListAsync()`** / **`CountAsync()`**.

---

## Why This Matters

`1.3.9`: pipelines wait. Production must **decide when to pull**. Too early — millions rows in RAM. Too late — duplicate SQL, disposed **`DbContext`**.

**Immediate** operators — trigger. API actions обычно заканчиваются **`ToListAsync()`** на shaped **`Select`**, не raw **`IQueryable`** в serializer.

## Core Concepts

### Terminal operators

| Category | Examples | Result |
|----------|----------|--------|
| Materialize | `ToList`, `ToArray`, `ToDictionary` | Collection |
| Aggregate | `Count`, `Sum`, `Max` | Scalar |
| Element | `First`, `Single`, `Last` | One item |
| Test | `Any`, `All` | `bool` |

### Materialization — snapshot

```csharp
List<User> snapshot = users.Where(u => u.IsActive).OrderBy(u => u.Name).ToList();
```

**`ToList()`** — full enumeration, independent copy. Source changes не affect snapshot.

### Count vs Any

```csharp
if (query.Any()) { ... }        // short-circuit
bool empty = query.Count() == 0; // full scan — хуже
```

**`users.Where(...).Count()`** — full scan. **`List.Count`** property — O(1).

### First / FirstOrDefault

**`First`** — throw if empty. **`FirstOrDefault`** — default if none.

### Multiple terminals = multiple executions

```csharp
var q = orders.Where(o => o.Open);
var count = q.Count();
var list = q.ToList();  // second scan
```

Prefer один **`ToList`**, then **`.Count`**.

### Web request boundary

```csharp
var dtos = await _db.Orders
    .Where(...)
    .Select(o => new OrderDto(...))
    .ToListAsync(ct);
return Ok(dtos);
```

Materialize после filter/project — до JSON. Не return **`IQueryable`** from controller.

---

## Under the Hood

**`ToList`**: allocate **`List<T>`**, foreach add. **`Any`/`First`** — short-circuit **`MoveNext`**.

---

## Common Mistakes

- **`Count()==0`** вместо **`!Any()`**
- Materialize до final **`Where`/`Select`**
- Return deferred **`IEnumerable`** from API

---

## Key Takeaways

- Terminal operators **end deferral**.
- **`ToList`** = snapshot, safe reuse.
- **`Any`** often cheaper than **`Count()`** for emptiness.
- One materialization per logical operation when possible.
