# Loops — краткая выжимка

> `1.1.10` · ~8 мин на повторение

## Главная идея

**Loops** повторяют block, пока не выполнится **termination condition**. Четыре формы — `for`, `while`, `do-while`, `foreach` — выбираются по тому, нужен ли **index**, известно ли **число итераций**, нужен ли **минимум один проход**.

## Ключевые пункты

- **`for`**: init; condition; iterator — counted loops, index `0 … Length-1`.
- **`while`**: condition до body; может быть 0 итераций.
- **`do-while`**: body минимум один раз; semicolon после `while (...)`.
- **`foreach`**: по `IEnumerable<T>`; без index; **не add/remove** во время обхода.
- **Off-by-one**: для arrays — `i < Length`, не `<= Length`.
- **Nested loops**: total iterations = произведение счётчиков.

## Самые важные примеры

```csharp
for (int i = 0; i < ids.Length; i++)
    Console.WriteLine(ids[i]);

foreach (string name in names)
    Console.WriteLine(name);
```

```csharp
do {
    Console.Write("Enter yes/no: ");
    input = Console.ReadLine();
} while (input != "yes" && input != "no");
```

## Частые ошибки

- Infinite loop — не обновили counter.
- Modify **List** в `foreach`.
- `for` без нужды в index вместо `foreach`.

## Мнемоника

> **for — считаю; while — жду; do — спрошу хотя бы раз; foreach — просто обойти.**
