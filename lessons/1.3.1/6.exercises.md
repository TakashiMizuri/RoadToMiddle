# Exercises: Generics — Classes, Methods, Constraints

> Complete after reading `1.lection-eng.md`, `2.lection-ru.md` and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Generic Stack
**Goal:** Implement a minimal `Stack<T>` with `Push`, `Pop`, `Count`, and `TryPeek`.
**Steps:** Create `Stack<T>` with internal array; double capacity when full; no constraints on `T`.
**Success criteria:** `Stack<int>` stores ints without boxing; `Pop` on empty throws or use `TryPop` pattern.

### E2. Constraint Exploration
**Goal:** Write `FindMax<T>(IEnumerable<T> items)` requiring `T : IComparable<T>`.
**Steps:** Iterate, track max with `CompareTo`; handle empty sequence (throw or return nullable).
**Success criteria:** Works for `int`, `string`, custom struct implementing `IComparable<T>`.

## Medium

### M1. Generic Repository Interface
**Goal:** Define `IReadRepository<TEntity>` and in-memory `InMemoryRepository<TEntity>` with `GetById(int id)`.
**Steps:** Constraint `where TEntity : class, IEntity` with `int Id { get; }`; store in `Dictionary<int, TEntity>`.
**Success criteria:** Compiler rejects struct entity; duplicate id on Add fails.

### M2. Factory with `new()`
**Goal:** `ActivatorFactory<T> where T : new()` with `Create()` and `CreateMany(int count)`.
**Steps:** Compare with `Activator.CreateInstance<T>()` (no constraint) — document difference in comment.
**Success criteria:** `CreateMany` returns `List<T>`; cannot use abstract class as `T`.

## Hard (stretch)

### H1. Open Generic Registration (Sketch)
**Goal:** Sketch ASP.NET Core registration for `IValidator<T>` → `FluentValidator<T>` (pseudo-code OK).
**Steps:** Show `services.AddTransient(typeof(IValidator<>), typeof(FluentValidator<>))` and constructor injection of `IValidator<Order>` in a handler.
**Success criteria:** Explain in one sentence how DI closes `T` at resolve time.

## Submission checklist

- [ ] No unnecessary constraints on type parameters
- [ ] Value type test (`Stack<int>`) passes without boxing (conceptual / dotMemory optional)
- [ ] Generic method type inference demonstrated in at least one call site
