# Exercises: Custom IEnumerable with yield

## Easy

### E1. Range iterator
**Goal:** `IEnumerable<int> Range(int from, int toInclusive)` using yield.
**Success criteria:** `foreach` works; no off-by-one.

### E2. Take early stop
**Goal:** Infinite naturals + `Take(3)` — verify only three iterations in debugger.
**Success criteria:** Observed MoveNext count ≤ 4.

## Medium

### M1. Paginated fake
**Goal:** Mock API returns pages of 10; yield flat stream; consumer `First()` fetches only first page.
**Success criteria:** Page 2 not requested in test spy.

### M2. ReadLines with disposal
**Goal:** Iterator reads temp file; break foreach mid-way — file handle closed (unit test or manual).
**Success criteria:** No file lock after break.

## Hard (stretch)

### H1. Batch yield
**Goal:** Implement `Batch<T>(this IEnumerable<T> source, int size)` yielding `IReadOnlyList<T>`.
**Success criteria:** Last batch partial; lazy until consumed.
