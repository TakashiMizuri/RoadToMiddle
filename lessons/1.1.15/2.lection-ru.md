# Collections intro: List, Dictionary, HashSet

> Roadmap: `1.1.15` · Node: `1.1` — C#: основы языка · Depth: практика

## Learning Objectives

После лекции ты сможешь:

- Использовать **`List<T>`** для ordered growable sequences с index access.
- Использовать **`Dictionary<TKey,TValue>`** для lookup по ключу (~O(1) average).
- Использовать **`HashSet<T>`** для uniqueness и set operations.
- Объяснить internal backing на высоком уровне.
- Выбирать collection по access pattern.
- Избегать **`KeyNotFoundException`**, modify-during-enum, wrong comparer.

---

## Why This Matters

Урок `1.1.14` — **arrays** (fixed length). Реальным приложениям нужны dynamic size, lookup by key, uniqueness. **`System.Collections.Generic`** — стандарт до specialized libraries.

Эти три типа встречаются в ASP.NET, EF, JSON DTO, deduplication. Middle знает **operations + complexity**: не сканировать **`List`** linearly, когда нужен **`Dictionary`**.

Общее: **generics**, **`IEnumerable<T>`**, **capacity growth**, **`IEqualityComparer<T>`**.

---

## Core Concepts

### List<T>

Оборачивает **`T[]`** + **`Count`**. **`Add`** amortized O(1); при заполнении capacity ~double + copy.

```csharp
var ids = new List<int> { 10, 20 };
ids.Add(30);
ids.Contains(20); // O(n)
```

**Когда:** order, duplicates, index by position. **Insert/remove middle** — O(n).

### Dictionary<TKey, TValue>

Hash table: average O(1) get/set; worst O(n) при collisions.

```csharp
var scores = new Dictionary<string, int> { ["alice"] = 100 };
scores.TryGetValue("bob", out int s);
```

**`dict[key]`** → **`KeyNotFoundException`** если нет. Prefer **`TryGetValue`**. Keys unique; **null keys** запрещены (default).

Modern **Dictionary** сохраняет insertion order при enumeration — не полагайся для correctness без документации.

### HashSet<T>

Unique elements через hashing.

```csharp
var seen = new HashSet<string> { "GET", "POST" };
seen.Add("GET"); // false
```

Set ops: `UnionWith`, `IntersectWith`, … **No indexing**, unordered для logic.

### Выбор

Order + duplicates + index → **List**. Lookup by key → **Dictionary**. Unique membership → **HashSet**. Часто **List + Dictionary** вместе.

### Equality

**Dictionary/HashSet** зависят от **`GetHashCode`/`Equals`**. **Mutable keys** — anti-pattern.

---

## Under the Hood

**List:** `T[]` + `_size`, grow double. **Dictionary/HashSet:** buckets + entries, rehash on load. **InvalidOperationException** при modify during **`foreach`**.

---

## Examples

Index from list: один pass → **`Dictionary<int, string>`**. Distinct tags: **`new HashSet<string>(tags, StringComparer.OrdinalIgnoreCase)`**. Safe read: **`TryGetValue`** без double lookup.

---

## Common Mistakes

`dict[key]` без check. List.Contains в hot lookup loop. Mutating keys. foreach + Add/Remove. Wrong string comparer.

---

## Production

Expose **`IReadOnlyList`/`IReadOnlyDictionary`**. Pre-size: **`new List<T>(n)`**, **`new Dictionary<K,V>(capacity)`**. Concurrency → **`ConcurrentDictionary`** (later).

---

## Key Takeaways

- **List** — dynamic array; **Dictionary** — map; **HashSet** — unique set.
- **TryGetValue** для safe get.
- Immutable keys для hash collections.
- Выбор по access pattern.
