## Answers

### 1. [Recall]
**Answer:** Object **still reachable** from a GC **root** though no longer needed logically.
**Why:** GC only frees unreachable objects.

### 2. [Recall]
**Answer:** Delegate stores **reference to `this`**; publisher (especially long-lived) keeps subscriber alive.
**Why:** MulticastDelegate target chain.

### 3. [Recall]
**Answer:** **Cannot `-=`** same delegate instance — subscription permanent.
**Why:** Unsubscribe requires identical delegate reference.

### 4. [Understanding]
**Answer:** **Static event field** (root) → delegate → **subscriber instance** → its fields graph.
**Why:** Static lives process lifetime.

### 5. [Understanding]
