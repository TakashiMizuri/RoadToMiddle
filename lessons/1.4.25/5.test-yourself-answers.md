## Answers

### 1. [Recall]
**Answer:** **.NET 6**.

**Why:** API introduced with modern parallel async support.

---

### 2. [Recall]
**Answer:** **-1**, interpreted as **`Environment.ProcessorCount`**.

**Why:** Default ParallelOptions behavior.

---

### 3. [Understanding]
**Answer:** Sync body blocks thread pool worker during I/O; often `.Result` on async → starvation; not async-aware scheduling.

**Why:** Wrong tool for I/O-bound work.

---

### 4. [Understanding]
**Answer:** WhenAll schedules/all tasks at once → 1000 concurrent connections/state; ForEachAsync caps active work → protects sockets, downstream, memory.

**Why:** Core lesson motivation.

---

### 5. [Application]
**Answer:**
```csharp
await Parallel.ForEachAsync(source, new ParallelOptions
{
    MaxDegreeOfParallelism = 12,
    CancellationToken = ct
}, async (item, token) => { await Work(item, token); });
```

**Why:** Standard pattern.

---

### 6. [Application]
**Answer:** **IHttpClientFactory** (not new client per call); forward **`CancellationToken`** / **RequestAborted**; consider max IDs validation; bounded parallelism.

**Why:** ASP.NET integration section.

---

### 7. [Analysis]
**Answer:** 8 concurrent may **under**-utilize I/O wait OR still overload fragile API; ProcessorCount irrelevant to network limits; 429, socket issues if downstream weak; should tune empirically (often > cores for I/O but capped by SLA).

**Why:** CPU count ≠ optimal HTTP parallelism.

---

### 8. [Analysis]
**Answer:** **`ConcurrentBag<T>`**, **`ConcurrentDictionary`**, lock on shared structure, or immutable fold — not plain **`List<T>.Add`** without sync.

**Why:** Parallel invocations race on mutable collections.

---

### 9. [Understanding]
**Answer:** Stop scheduling new iterations; in-flight observe token → **`OperationCanceledException`**; loop may end incomplete (`IsCompleted` false).

**Why:** Cooperative cancellation model.

---

### 10. [Application]
**Answer:** When need explicit batch boundaries, different limits per chunk, or memory control for huge N — `Chunk(size)` + WhenAll inside sequential chunk loop.

**Why:** Example 2 in lecture.

---

## Model answers for practical

### P1
**Answer:** ForEachAsync urls, MaxDegreeOfParallelism=5, try GetAsync, catch 404 add to failureBag, other exceptions optionally rethrow or add; use factory client; pass ct.

**Why:** Combines bound, resilience, thread-safe aggregation.
