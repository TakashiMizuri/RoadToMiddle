# LINQ Any, All, Contains — краткая выжимка

> `1.3.24` · ~7 мин

## Главная идея

**`Any`** — существует ли element / match; **`All`** — все ли satisfy; **`Contains`** — membership по equality. Все **short-circuit** (кроме worst case full walk).

## Ключевые пункты

- **`Any()`** вместо **`Count() > 0`**.
- **`All`** на **empty → true** (vacuous truth).
- Validation: **`Any() && All(rule)`** для non-empty + all valid.
- Auth: **`!Contains("Admin")`**, не **`All(r => r != "Admin")`** на empty.
- **`Contains(value, comparer)`** для case-insensitive.
- EF: **`AnyAsync`** → EXISTS-style.

## Примеры

```csharp
orders.Any(o => o.IsPaid);
lines.Any() && lines.All(l => l.Qty > 0);
roles.Contains("Admin", StringComparer.OrdinalIgnoreCase);
```

## Частые ошибки

- Vacuous **`All`** без **`Any()`** guard.
- **`CountAsync > 0`** вместо **`AnyAsync`**.
