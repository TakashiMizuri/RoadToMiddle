# Delegates: multicast — краткая выжимка

> `1.3.3` · ~10 мин

## Главная идея

**Delegate** — type-safe указатель на method (+ target object). **`+=`** строит **multicast** chain; **`Invoke`** вызывает все handlers по порядку. **`-=`** удаляет по совпадению target+method — не по «логической» equality lambdas.

## Ключевые пункты

- Custom `delegate void Del()` vs `Action` — naming vs convenience.
- Multicast **immutable** — каждый `+=` новый объект.
- **`Func` multicast** — return только **последнего** handler.
- Exception в середине — later handlers могут не run.
- **`GetInvocationList()`** — snapshot для safe iteration.
- **`event`** — защита от external `=` wipe.

## Примеры

```csharp
Notify n = OnData;
combined += n;
combined -= n; // OK

combined += (s) => Log(s);
combined -= (s) => Log(s); // FAIL — different compiler method
```

## «Под капотом»

`MulticastDelegate` хранит array delegates. `Invoke` — loop. `Delegate.Combine`/`Remove` — BCL API для `+=`/`-=`.

## Частые ошибки

- Unsubscribe новой lambda.
- Думать, что returns суммируются.
- Public `Action` field вместо event.
