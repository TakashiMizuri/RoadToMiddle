# Exercises: Constructors

> Complete after lectures and passing `4.test-yourself.md`.

## Easy

### E1. Chain overloads
**Goal:** `: this()` DRY.

`Square` from `Rectangle` pattern: ctor `(double side)` chains to `(width, height)` with positive validation.

**Success criteria:** One validation block.

---

### E2. Missing parameterless
**Goal:** Compiler rule.

Class with only `(int id)` ctor — show `new C()` fails; add explicit parameterless that chains `: this(0)`.

**Success criteria:** Both compile after fix.

---

## Medium

### M1. Primary constructor service
**Goal:** DI-style type.

`NotificationService(IEmailSender sender)` with method `Notify(string to, string body)`.

**Success criteria:** Constructor injection without manual field boilerplate.

---

### M2. Factory + private ctor
**Goal:** Controlled creation.

`EmailAddress` only via `Create(string)` validating `@`; private ctor.

**Success criteria:** External `new EmailAddress` impossible.

---

## Hard (stretch)

### H1. Init + ctor together
**Goal:** required + constructor.

Class with `required Guid Id { get; init; }` and ctor assigning Id with extra validation. Document legal creation paths.

**Success criteria:** Lists ctor-only and ctor+initializer paths.
