# IHttpClientFactory — named clients, typed clients, handlers

> Roadmap: `1.4.27` · Node: `1.4` — Async + ASP.NET preview · Depth: **глубоко**

## Learning Objectives

После урока ты сможешь:

- Регистрировать **named HttpClients** через **AddHttpClient** / **CreateClient(name)**.
- Строить **typed clients** (`AddHttpClient<TInterface, TImpl>`).
- Chain **DelegatingHandler** для logging, auth, correlation ID, resilience.
- Объяснить **handler lifetime** (~2 min) и влияние на DNS/pool.
- Настраивать **ConfigurePrimaryHttpMessageHandler**.
- Безопасно использовать factory с **Parallel.ForEachAsync**.

---

## Why This Matters

**IHttpClientFactory** решает lifetime problem (`1.4.26`) через DI. Каждый downstream — named config: BaseAddress, timeout, headers, Polly (`1.4.28`), handler chain. Typed wrappers testable с **WebApplicationFactory**. **DelegatingHandler** = outbound middleware.

Misconfig: wrong handler order, captive dependency scoped в singleton handler, static HttpClient рядом с factory.

---

## Core Concepts

### Named client

```csharp
builder.Services.AddHttpClient("catalog", c =>
{
    c.BaseAddress = new Uri(config["Catalog:BaseUrl"]!);
});

var client = factory.CreateClient("catalog");
```

### Typed client

```csharp
builder.Services.AddHttpClient<ICatalogApi, CatalogApi>(...);
// Inject ICatalogApi
```

### Handler lifetime

Default **2 minutes** rotation → DNS refresh. **SetHandlerLifetime** для tune.

### DelegatingHandler

```csharp
builder.Services.AddHttpClient("catalog")
    .AddHttpMessageHandler<CorrelationIdHandler>()
    .AddHttpMessageHandler<AuthTokenHandler>();
```

Outer handler — first on outgoing request. Polly inserts here.

### Primary handler

**ConfigurePrimaryHttpMessageHandler** — SocketsHttpHandler: PooledConnectionLifetime, MaxConnectionsPerServer.

---

## Under the Hood

Factory manages **handler entries**, не pool HttpClient instances. **CreateClient** cheap. Captive dependency: scoped в singleton handler — use **IServiceScopeFactory** в SendAsync.

---

## Common Mistakes & Anti-patterns

Static HttpClient. Wrong handler order. One default client для всех downstreams. HostedService + scoped client без scope.

---

## Key Takeaways

- Factory manages **handlers**, DNS rotation.
- Named per downstream; typed для DI/tests.
- DelegatingHandler pipeline.
- Foundation для Polly `1.4.28`.
