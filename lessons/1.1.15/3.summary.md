# Collections intro: List, Dictionary, HashSet — краткая выжимка

> `1.1.15` · ~7 мин на повторение

## Главная идея

**`List<T>`** — динамический массив с порядком и дубликатами; **`Dictionary<K,V>`** — hash map для lookup по ключу; **`HashSet<T>`** — уникальные элементы и set operations.

## Ключевые пункты

- **List:** `Add` O(1) amortized; `Contains` O(n); index `list[i]` O(1).
- **Dictionary:** `TryGetValue` — один lookup; `[key]` бросает **`KeyNotFoundException`** если нет ключа.
- **HashSet:** уникальность через hash; `Add` возвращает `false` для duplicate.
- Ключи dictionary/set: consistent **`GetHashCode`/`Equals`**; **не mutировать key** после insert.
- Нельзя **менять collection во время `foreach`** (кроме safe patterns).
- Часто **комбинируют**: List для порядка + Dictionary для lookup by id.

## Самые важные примеры

```csharp
var list = new List<int> { 1, 2 };
list.Add(3);

var map = new Dictionary<string, int>();
if (map.TryGetValue("key", out int v)) { }

var set = new HashSet<string> { "a", "b" };
set.Add("a"); // false
```

## Частые ошибки

- `dict[key]` без проверки.
- Linear scan List вместо Dictionary для частых lookup.
- Mutable key object после добавления в hash collection.
- Ожидать stable order у HashSet для business logic.

## Мнемоника

**List** = **Line** (ordered sequence). **Dictionary** = **Definition** (key → meaning). **HashSet** = **Unique bag**.
