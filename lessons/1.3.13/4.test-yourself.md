### 9. [Application]
Какой arm сработает для `args = new[] { "delete", "42" }`?

```csharp
args switch
{
    ["create", var name] => "create",
    ["delete", var id] when int.TryParse(id, out _) => "delete",
    ["list"] => "list",
    _ => "unknown"
};
```
