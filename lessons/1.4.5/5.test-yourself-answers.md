# Answers: TaskCompletionSource

### 1. Bridge external callbacks/events to awaitable Task when no async API exists.
### 2. SetResult throws InvalidOperationException; TrySetResult returns false.
### 3. Create TCS, subscribe handler TrySet*, await tcs.Task, unsubscribe in finally/continuation.
### 4. Avoid running await continuations synchronously on callback thread — deadlock/stack risk.
### 5. OK in race (first wins); second callback ignored.
### 6. CancellationTokenSource + Register TrySetCanceled; await WhenAny with delay task optional.
### 7. WaitingForActivation (or Created depending on ctor).
### 8. FromResult — already have value sync; TCS — completion happens later from external signal.
