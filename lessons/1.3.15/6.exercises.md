# Exercises: Attributes and Reflection

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Custom attribute
**Goal:** `[Unit("ms")]` on property; reader prints unit name.
**Steps:** Define attribute with string property; loop properties with `GetCustomAttribute<UnitAttribute>()`.
**Success criteria:** Console shows unit for annotated properties only.

### E2. Obsolete warning
**Goal:** Mark method `[Obsolete("Use B")]`; observe compiler warning CS0618.
**Success criteria:** Document warning in comment.

### E3. typeof vs GetType
**Goal:** Explain difference for `string` instance vs static `typeof(string)`.
**Success criteria:** One paragraph in README or comment.

## Medium

### M1. Mini validator
**Goal:** Attributes `[Required]`, `[MaxLength(n)]` on DTO; `Validate(object)` returns error list via reflection.
**Success criteria:** No external validation library; uses `GetCustomAttribute`.

### M2. Method catalog
**Goal:** Reflect all public methods on a class; print those with custom `[DisplayName("...")]`.
**Success criteria:** `BindingFlags.Public | Instance | DeclaredOnly`.

### M3. Invoke calculator
**Goal:** Class with `Add(int,int)`; locate via `GetMethod`, call via `Invoke`.
**Success criteria:** Result matches direct call; note performance in comment.

## Hard (stretch)

### H1. Attribute without instantiation
**Goal:** Use `CustomAttributeData.GetCustomAttributes` on assembly; read constructor argument without creating attribute instance.
**Success criteria:** Print value from metadata only.

## Submission checklist

- [ ] Custom attribute with `[AttributeUsage]`
- [ ] At least one `GetCustomAttribute<T>()` call
- [ ] Used `nameof` for member lookup
