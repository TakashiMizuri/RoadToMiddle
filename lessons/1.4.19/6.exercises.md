# Exercises: Exception Handling in Async

> Complete after passing `4.test-yourself.md`.

## Easy

### E1. Await vs Result
**Goal:** Feel AggregateException.

**Steps:** `Task t = Task.FromException(new InvalidOperationException("x"));` Try `await t` in try/catch — note type. New run: try `t.Wait()` — note type.

### E2. throw preservation
**Goal:** Stack traces.

**Steps:** Async method throws deep. Catch with `throw;` vs `throw ex;` — compare stacks in debugger (describe difference in comment).

---

## Medium

### M1. WhenAll reporter
**Goal:** Log all faults.

**Steps:** Start 5 tasks, 3 throw different exceptions. `await WhenAll` with catch that logs **all** fault messages.

### M2. ASP.NET mapping
**Goal:** Translate exceptions.

**Steps:** Sketch `async Task<ActionResult>` that catches `NotFoundException` → 404, `ValidationException` → 422, lets others bubble.

---

## Hard (stretch)

### H1. Safe fire-and-forget
**Goal:** Observe unobserved.

**Steps:** Describe pattern: extension `Forget()` with ContinueWith logging — when acceptable in web app vs **`1.4.30`**.
