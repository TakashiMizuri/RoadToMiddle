# Exercises: Span and Memory

## Easy

### E1. Parse line
**Goal:** Split `ReadOnlySpan<char>` by `,` without `string.Split`; parse ints.
**Success criteria:** No Substring in loop.

### E2. Stack buffer UTF-8
**Goal:** `stackalloc byte[64]` + `Encoding.UTF8.GetBytes(ReadOnlySpan<char>, Span<byte>)`.
**Success criteria:** Explain max stack size risk if 64 → 1_000_000.

## Medium

### M1. Trim helper
**Goal:** `Trim(ReadOnlySpan<char>)` returns slice; unit test mutates backing only via Span path on char[].
**Success criteria:** Zero alloc in benchmark loop (conceptual).

### M2. Memory async write
**Goal:** `byte[]` → `ReadOnlyMemory<byte>` → `MemoryStream.WriteAsync`.
**Success criteria:** Works with await.

## Hard

### H1. SequenceReader intro
**Goal:** Read two fake frames from `ReadOnlySequence<byte>` using `SequenceReader`.
**Success criteria:** Diagram buffer vs single array.
