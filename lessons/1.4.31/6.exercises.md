# Exercises: UnobservedTaskException

> Complete after passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Observation Checklist
**Goal:** For 4 code snippets (await, discard, `.Exception`, `Wait()`), mark observed vs unobserved.
**Success criteria:** 4/4 with one-line why.

### E2. Startup Registration
**Goal:** Add `UnobservedTaskException` handler to a console or web `Program.cs` skeleton.
**Success criteria:** Single registration; uses `SetObserved()`.

## Medium

### M1. Repro Console
**Goal:** Small console app: fire-and-forget fault; subscribe to event; observe delayed log.
**Success criteria:** Explain GC timing; force `GC.Collect()` for demo only.

### M2. Filter Cancel
**Goal:** Extend handler to skip `OperationCanceledException` but alert on others.
**Success criteria:** Metric or log level distinction documented.

## Hard (stretch)

### H1. Test Harness
**Goal:** xUnit test that fails if unobserved exception fires during test run.
**Success criteria:** Subscribe in fixture; `SetObserved()` + assert fail on non-cancel.
