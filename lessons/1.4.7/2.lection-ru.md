## Core Concepts

### IAsyncStateMachine

```csharp
void MoveNext();
void SetStateMachine(IAsyncStateMachine stateMachine);
```

**MoveNext** — весь async body как **stateful procedure**.

**SetStateMachine** — box struct на heap для stable continuation reference.

### Generated shape

Switch по **`<>1__state`**: **default** (start) → await #1 → **case 0** → await #2 → **case 1** → **SetResult**.

**AwaitUnsafeOnCompleted** — register **MoveNext**; return; thread free.

**State lifecycle:** `-1` start, `0..N` resume, `-2` finished.

### SetResult / SetException

Success → **SetResult**. Catch → **SetException**. Без Set* после suspend — in progress; never Set* — hung Task.

## Examples

### Пример 1: Single await — minimal machine

States: **default** → optional **case 0** → **SetResult**. Одно поле **TaskAwaiter**.

### Пример 2: Exception до первого await

Throw в **default** до **AwaitOnCompleted** → catch → **SetException** — suspension не было.

### Пример 3: await в loop / foreach await

Compiler генерирует loop внутри **MoveNext** с сохранением state — сложнее (`IAsyncEnumerable`, `1.4.16`).

## Common Mistakes & Anti-patterns

**Редактировать generated files** — только source.

**Stable state numbers** — compiler version может менять; не hardcode.

**Confuse builder.Task** (return caller) с inner awaited Task в nested async.

---

## Production & Real-World Notes

Task never completes — hung **MoveNext**? missing IO callback? TCS never set? SyncContext deadlock?

Reduce await count в hot paths; **ValueTask**; sync fast path без state machine.

State machine fields держат captured locals до GC — sensitive data как closures.

## Key Takeaways

- **MoveNext** = твой async method split at awaits.
- **AwaitUnsafeOnCompleted** + state = thread-free I/O.
- **SetResult/SetException** завершают Task caller.
- Далее: **ConfigureAwait** (`1.4.9`), deadlocks (`1.4.14`).
