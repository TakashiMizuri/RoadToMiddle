# OrderBy, ThenBy — stable sort — краткая выжимка

> `1.3.28` · ~6 мин

## Главная идея

**`OrderBy`** — primary sort; **`ThenBy`** — secondary. Второй **`OrderBy`** сбрасывает первый. LINQ to Objects — **stable sort** (equal keys сохраняют input order).

## Ключевые пункты

- Multi-key: `OrderBy(...).ThenBy(...).ThenByDescending(...)`.
- Pagination: always `ThenBy(uniqueId)` для deterministic pages.
- EF: `ORDER BY` на server; sort before `Skip/Take`.
- `AsEnumerable()` before sort → client-side, often full table pull.
- Custom: `StringComparer`, `Comparer.Create`.

## Примеры

```csharp
.OrderByDescending(s => s.Points).ThenBy(s => s.SubmittedAt);
.OrderBy(i => i.SortOrder).ThenBy(i => i.Id);
```

## Частые ошибки

- Chain `OrderBy` вместо `ThenBy`.
- Нет tie-breaker при paging.
