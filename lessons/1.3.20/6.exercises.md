# Exercises: yield return — Iterator Pattern

> Complete after lectures and passing `4.test-yourself.md`. Ask AI for hints, not full solutions.

## Easy

### E1. Trace enumeration
**Goal:** Observe lazy execution order.

**Tasks:** Predict output order, then run:
```csharp
static IEnumerable<int> Gen()
{
    Console.WriteLine("A");
    yield return 1;
    Console.WriteLine("B");
    yield return 2;
}
var seq = Gen();
Console.WriteLine("C");
foreach (var x in seq) Console.WriteLine(x);
```

**Success criteria:** Written prediction matches output; explain each line.

---

### E2. yield break
**Goal:** Early termination.

**Tasks:** Implement **`TakeWhilePositive`** — yield positive ints from source until first `<= 0`, then stop without consuming rest into a list.

**Success criteria:** Works on `[3,1,-1,5]` → `3,1` only.

---

## Medium

### M1. ReadLines with disposal
**Goal:** Resource lifetime tied to enumeration.

**Tasks:** Write **`ReadFirstNLines(string path, int n)`** using **`yield`**. Verify file handle released when stopping after **`n`** lines (use **`FileShare.None`** test file or logging in **`finally`**).

**Success criteria:** Early stop does not leak handle; full read disposes.

---

### M2. Custom Where
**Goal:** Build deferred operator.

**Tasks:** Implement **`WhereLazy<T>(this IEnumerable<T> source, Func<T,bool> pred)`** with **`yield`**. Show **`pred`** is not called until enumeration.

**Success criteria:** Demo with side-effect predicate called zero times before **`foreach`**.

---

## Hard (stretch)

### H1. Chunk lazy pipeline
**Goal:** Compose lazy stages.

**Tasks:** **`Chunk(size)`** + **`WhereLazy`** over **`Range(1,10)`** from **`yield`**. Materialize only final chunk list; count predicate invocations.

**Success criteria:** Memory not O(full source); documented invocation count.
