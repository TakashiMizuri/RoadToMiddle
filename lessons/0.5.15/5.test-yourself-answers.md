## Answers

### 1. [Recall]
**Answer:** **`data-`**
**Why:** HTML spec reserves `data-*` for site-specific metadata; guarantees no future standard collision.

### 2. [Recall]
**Answer:** **`dataset.userId`**
**Why:** Hyphens after `data-` become camelCase; first segment `user` + `Id`.

### 3. [Recall]
**Answer:** Custom Elements, Shadow DOM, HTML `<template>` (templates).
**Why:** Three pillars of Web Components platform APIs.

### 4. [Understanding]
**Answer:** `dataset.stock` is string **`"12"`**; strict equality `=== 12` (number) is false.
**Why:** All data attribute values expose as DOMString; need `Number()` or `==` with coercion awareness.

### 5. [Understanding]
**Answer:** Avoid collision with current/future **built-in HTML tag names**; spec requires hyphen in custom tag names.
**Why:** Parser distinguishes custom from standard elements.

### 6. [Understanding]
**Answer:** `data-*` is **valid** in HTML5 and forward-compatible; arbitrary attrs fail validation and may be dropped.
**Why:** Spec-defined extension point vs non-conforming markup.

### 7. [Application]
**Answer:**
```html
<button type="button" data-action="delete" data-item-id="7">Delete</button>
```
```javascript
const id = btn.dataset.itemId; // "7"
```
**Why:** kebab-case in HTML, camelCase in dataset API.

### 8. [Application]
**Answer:** When row data must **submit with a form** — hidden inputs participate in form data; `data-*` does not.
**Why:** Form encoding only includes named form controls.

### 9. [Analysis]
**Answer:** **`enableFeature()` runs** — non-empty string `"false"` is **truthy** in JavaScript.
**Why:** Common bug; need explicit compare `dataset.enabled === 'true'` or parse boolean.

### 10. [Understanding]
**Answer:** Lifecycle hook when element **inserted into document** — setup, render, listeners.
**Why:** Analog mount in component frameworks; runs each time connected to DOM tree.

### 11. [Analysis]
**Answer:** Missing native button semantics (role, keyboard activation default); worse accessibility and behavior vs `<button>`; tabindex hack fragile.
**Why:** `0.5.11` — use native interactive elements; data-* does not fix semantics.

### 12. [Application]
**Answer:** **`await customElements.whenDefined('product-badge')`** or callback form before interacting.
**Why:** Definition may run after parser created unknown elements; API waits for upgrade.
