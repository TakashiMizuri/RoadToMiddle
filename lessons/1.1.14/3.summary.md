# Массивы: одномерные, jagged, многомерные — краткая выжимка

> `1.1.14` · ~7 мин на повторение

## Главная идея

**Array** в C# — reference type фиксированной длины с zero-based индексами. Форма данных задаёт тип: **`T[]`** (линия), **`T[,]`** (прямоугольная сетка), **`T[][]`** (массив массивов с разной длиной строк).

## Ключевые пункты

- **`T[]`**: индекс `a[i]`, длина `a.Length`.
- **`T[,]`**: индекс `a[row, col]`; **`GetLength(0)`** / **`GetLength(1)`** — размеры измерений; **`Length`** = произведение (все ячейки).
- **`T[][]`**: индекс `a[i][j]`; outer `Length` ≠ длина строки; каждую строку нужно **`new`** отдельно.
- Rectangular — **один contiguous block** (row-major); jagged — **отдельные row-массивы** на heap.
- **`foreach`** просто по `int[,]` без вложенных циклов — нет; по `T[]` — да.
- Array **covariance** только для reference types — опасные runtime cast при записи.

## Самые важные примеры

```csharp
int[] one = { 1, 2, 3 };
int[,] grid = new int[3, 4];
grid[1, 2] = 7;

int[][] jagged = new int[2][];
jagged[0] = new int[] { 1, 2 };
jagged[1] = new int[] { 3 };
```

## Частые ошибки

- `grid[1][2]` вместо `grid[1, 2]`.
- Считать `grid.Length` числом строк.
- NullReference на jagged: строка не создана.
- Массив вместо `List<T>` при частых insert/remove.

## Мнемоника / правило большого пальца

**Фиксированная таблица** → `[,]`. **Строки разной длины** → `[][]`. **Одна колонка/линия** → `[]`.
