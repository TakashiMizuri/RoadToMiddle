# Immediate execution — краткая выжимка

> `1.3.10` · ~6 мин на повторение

## Главная идея

**Terminal operators** (`ToList`, `Count`, `First`, `Any`, `Sum`, …) и **`foreach`** **запускают** deferred pipeline. **`ToList()`** создаёт **snapshot** — стабильный список, безопасный для повторного использования.

## Ключевые пункты

- **Materialize:** `ToList`, `ToArray`, `ToDictionary`, `ToHashSet`
- **Scalar / pick:** `Count`, `Sum`, `First`, `Single`, …
- **Existence:** `Any` (short-circuit) vs `Count()==0` (full scan)
- **`Where(...).Count()`** — always scan matches; **`list.Count`** — O(1)
- Два terminal на одном `q` → два execution (`1.3.9`)
- Materialize **once** at API boundary; EF: **`ToListAsync`**

## Самые важные примеры

```csharp
var page = query.OrderBy(x => x.Id).Skip(n).Take(size).ToList();
```

```csharp
var list = await db.Users.Where(u => u.Active).ToListAsync();
return Ok(list);
```

## Частые ошибки

- Double **`CountAsync` + `ToListAsync`** без need
- Materialize entire table before **`Where`**
- **`First`** when absence is normal → use **`FirstOrDefault`**

## Мнемоника

> **Defer to build, terminal to run** — один terminal на logical operation.
