# Exercises: IHttpClientFactory

> Complete after lectures and test.

## Easy

### E1. Named client registration
**Goal:** Register "geo" client with BaseAddress from appsettings.
**Steps:** AddHttpClient + controller/Minimal API GET proxy.
**Success criteria:** Config-driven URL change works without recompile.

### E2. CorrelationId handler
**Goal:** Outbound requests include X-Correlation-ID from TraceIdentifier.
**Steps:** DelegatingHandler + AddHttpMessageHandler; verify with httpbin.org/headers or local echo.
**Success criteria:** Header present on downstream request.

## Medium

### M1. Typed client
**Goal:** ICatalogApi with GetItemAsync; inject into endpoint.
**Steps:** Interface + implementation taking HttpClient; AddHttpClient registration.
**Success criteria:** Unit test with Mock<HttpMessageHandler> returns fake JSON.

### M2. Handler order experiment
**Goal:** Demonstrate auth-before-retry requirement.
**Steps:** Two handlers logging order; optional simulated 401 without token on inner retry.
**Success criteria:** Document correct registration order.

## Hard (stretch)

### H1. HostedService with scope
**Goal:** Periodic sync using typed client from singleton BackgroundService.
**Steps:** IServiceScopeFactory pattern; cancellation on shutdown.
**Success criteria:** No captive dependency validation errors at startup.
